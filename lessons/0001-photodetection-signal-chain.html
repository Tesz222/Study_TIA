📡 光电信号链

Lesson 0001 · 光电检测 · 预计用时 20 分钟

💡 本节核心 理解从光信号到数字信号的完整转换链路：
光 → 电流（光电二极管）→ 电压（跨阻放大器 TIA）→ 数字（ADC 采样）

1. 第一环：光 → 电流（光电二极管）

1.1 光电效应基础

当光子能量 E = hν 大于半导体材料的禁带宽度 Eg 时，光子被吸收并产生电子-空穴对。这就是 内光电效应（photoelectric effect）。

E = h · f = hc / λ

其中：
  h  = 6.626×10⁻³⁴ J·s  （普朗克常数）
  c  = 3×10⁸ m/s        （光速）
  λ  = 波长（m）
  f  = 频率（Hz）

1.2 关键参数

参数符号说明典型值
响应度R单位光功率产生的光电流0.3~0.6 A/W @ 635nm
暗电流ID无光照时的漏电流nA ~ pA 级
结电容CJ影响频率响应速度2~50 pF
带宽fc可用频率上限与 CJ 和负载相关
响应波长范围λ能有效响应的光谱区间Si: 400~1100 nm
🔑 关键认知 光电二极管的输出是 电流，不是电压。这个电流通常非常微弱（μA ~ nA 级），不能直接被 ADC 采样，必须经过 I-V 转换。

2. 第二环：电流 → 电压（跨阻放大器 TIA）

2.1 为什么需要 TIA

光电二极管的输出是电流，但 ADC 需要电压输入。我们需要一个 跨阻放大器（Transimpedance Amplifier, TIA）来完成 I-V 转换。其核心是一个运放 + 反馈电阻的结构：

2.2 TIA 工作原理

运放的"虚短"（virtual short）特性使反相输入端保持在地电位（同相端接地）。因此光电流 IPD 全部流过反馈电阻 Rf，在输出端产生电压：

Vout = -IPD x Rf

举例：
  IPD = 10 μA    （中等光照下的光电流）
  Rf  = 100 kΩ   （反馈电阻）

  Vout = -10 μA x 100 kΩ = -1 V
📐 设计要点
  • Rf 决定增益：Rf 越大，相同光照下输出电压越高
  • Cf 并联在 Rf 上用于相位补偿，防止振荡
  • 运放选择需关注：输入偏置电流（应远小于 IPD）、GBW（增益带宽积）

2.3 噪声考虑

TIA 电路中主要的噪声来源：

  • 反馈电阻热噪声：vn² = 4kTRf — Rf 越大，噪声越大
  • 运放电压噪声：被电路增益放大
  • 运放电流噪声：流过 Rf 产生电压噪声
🔑 关键认知 TIA 是整个信号链的"心脏"——它决定了信噪比和带宽。增益（Rf）和带宽之间存在根本性的 trade-off：高增益 → 大 Rf → 更多热噪声 + 更窄带宽。

3. 第三环：模拟电压 → 数字信号（ADC 采样）

3.1 ADC 的核心概念

模数转换器（Analog-to-Digital Converter, ADC）将连续的模拟电压离散化为有限比特的数字量。

参数符号说明
分辨率N bits将满量程分成 2N 个量化层级
参考电压VREFADC 的满量程电压范围
量化步长（LSB）VLSBVREF / 2N — 能分辨的最小电压变化
采样率fs每秒采样的次数 (SPS)
数字输出码 = round( Vin / VREF x 2N )

举例（12-bit ADC, VREF = 3.3 V）：
  Vin = 1.0 V
  数字码 = round(1.0 / 3.3 x 4096) = round(1241.2) = 1241
  二进制：0100 1101 1001

3.2 采样定理（Nyquist）

⚠️ 必须遵守 采样率 fs 必须 ≥ 2 x 信号最高频率 fmax，否则会发生 混叠（aliasing），即高频成分折叠到低频段，产生无法恢复的失真。

3.3 ADC 常见架构

类型速度精度典型应用
逐次逼近型 SAR中速 (MSPS)8~18 bit工业采集、PLC 模拟量输入
Σ-Δ（Delta-Sigma）低速 (~kSPS)16~24 bit高精度测量、称重传感器

5. 动手练习：参数计算

📝 练习 1：你有一个光电二极管，响应度 R = 0.5 A/W，接收到 50 μW 的光功率。你使用 TIA 的 Rf = 200 kΩ。求：

  1. 光电流 IPD 是多少？
  2. TIA 输出电压 Vout 是多少？
  3. 如果使用 12-bit ADC，VREF = 3.3 V，这个电压对应的数字码是多少？
答案：
① IPD = 0.5 x 50x10⁻⁶ = 25 μA
② Vout = 25 μA x 200 kΩ = 5 V（注意 5V 已超 3.3V 参考电压，需要降低增益或使用分压/衰减）
③ 如果实际输入 ADC 为 2.5 V（经衰减），数字码 = round(2.5 / 3.3 x 4096) = 3103

📝 练习 2：概念检查

为什么光电二极管输出的是电流而不是电压？TIA 中运放的作用是什么？

答案：光电二极管本质上是一个电流源——光生载流子在外加反偏电压下形成漂移电流。TIA 中的运放利用"虚短"特性将输入电流精确地引导到反馈电阻上，从而产生与电流成正比的电压输出。如果没有 TIA，直接用电阻负载测量电压会受结电容影响，导致严重的带宽和线性度问题。
