任务 1.1：启动 OrCAD Capture 并选对 Product Choice

▶ 1.1.1 找到 OrCAD Capture CIS

Cadence 17.4 在桌面没有快捷方式，必须从开始菜单启动。
☐ 点击 Windows 开始 按钮（或按键盘 Windows 键）
☐ 在搜索框中输入 Capture CIS
☐ 看到 OrCAD Capture CIS 或 Capture CIS 图标后，点击打开
☐ 如果找不到，在开始菜单中找 Cadence 文件夹：Cadence → Release 17.4 → OrCAD Capture CIS

⚠ 1.1.2 第一次启动：Product Choices 弹窗

第一次打开时，会弹出一个叫 Cadence Product Choices 的窗口，里面列了很多选项。
这是 Cadence 让你选择以哪个产品模式运行——原理图、仿真、还是 PCB 布局。
☐ 在列表中找到 "OrCAD Capture CIS" 这一行（列表可能很长，往下翻）
☐ 点击选中它
☐ 在窗口左下方找到 Use as default 复选框，勾选上
☐ 点击 OK
💡 勾选 Use as default 后，下次打开就不会再弹这个窗口了。如果想换，看下一步。

▶ 1.1.3 验证：确认进入了 OrCAD Capture 主界面

☐ 确认看到以下界面元素：
  • 顶部：菜单栏（File / Edit / View / Place / Tools / Reports / Help）
  • 左侧：项目管理器（Project Manager）树形结构
  • 中间：灰色的原理图编辑区域（目前是空的）
  • 底部：Session Log 窗口
💡 如果界面全是英文，不要慌——Cadence 业界标准就是英文界面，中文版反而查不到资料。

▶ 1.1.4 学会切换产品特性（非常重要）

如果哪天你需要用到 PSpice 仿真功能，或者想从 OrCAD 模式切换到 Allegro 模式，不需要重装软件。
☐ 在 OrCAD 菜单栏上点击 File → Change Product
☐ 你会再次看到 Product Choices 窗口（和第一次启动时一样）
☐ 浏览一下各选项，找到 "OrCAD PSpice Designer" 在哪一行
☐ 不要切换，点击 Cancel 退出
💡 记住这个路径：File → Change Product。以后如果你需要仿真或切换 Allegro 模式，就用它。

任务 1.2：启动 Allegro PCB Editor

▶ 1.2.1 找到 Allegro PCB Designer

☐ 开始菜单 → 搜索 Allegro PCB Designer
☐ 或找：Cadence → Release 17.4 → PCB Editor

⚠ 1.2.2 Product Choices（Allegro 版）

☐ 弹窗后，选择 Allegro PCB Designer
☐ 在 Options 区域中，把能勾的全勾上（避免后续提示 License 不足）
☐ 勾选 Use as default
☐ 点击 OK

▶ 1.2.3 浏览 Allegro 主界面

☐ 确认进入 Allegro PCB Editor 主界面
☐ 浏览菜单栏：Setup、Display、Manufacture、Tools——这几个是以后最常用的
☐ 观察右侧的 Options 面板和底部的命令栏（以后会经常看这里）
☐ 观察完毕后，直接关掉——不需要保存（File → Exit，不保存）

任务 1.3：快捷键初识（最重要的 8 个键）

Cadence 设计哲学：键盘比鼠标快 3~5 倍。高手不用点菜单，全程键盘。今天先记住 8 个最基础的。

▶ 1.3.1 回到 OrCAD 并新建一个空白页

☐ 如果 OrCAD 刚才关掉了，重新打开（已勾选 Use as default，不会弹窗了）
☐ 按 Ctrl+N（新建）→ 在 File Name 中输入 EMPTY → OK
☐ 按 I（放大）几次，看到灰色编辑区域
按 I 是 Zoom In（放大），按 O 是 Zoom Out（缩小）。和一般软件不一样，OrCAD 用 I/O 而不是 +/-。

🎯 1.3.2 练习 8 个快捷键（每个按键重复 3 遍）

第一组：放置类（P / W / F / G / N）

☐ 按 P → 弹出 Place Part 窗口 → 随便选一个器件 → 放下来 → 按 Esc 退出
P = Place Part（放置器件）。记住：P = Part。
☐ 按 W → 光标变成十字 → 在页面上点两下画一条线 → 按 Esc 退出
W = Place Wire（走线/导线）。记住：W = Wire。
☐ 按 F → 弹出一个窗口 → 选 VCC 符号 → 放下来 → Esc
F = Place Power（电源符号）。不用记理由，记住 F 就行。
☐ 按 G → 选 GND（或 0/SOURCE）符号 → 放下来 → Esc
G = Place Ground（地符号）。G = Ground。
☐ 按 N → 弹出 Net Alias 窗口 → 输入 TEST → 放在一根线上 → Esc
N = Place Net Alias（网络别名/网标）。N = Net。

第二组：导航和编辑类（I / O / R）

☐ 按 I（放大）→ 按 O（缩小）→ 重复 3 次
☐ 选中一个器件 → 按 R（旋转）→ 看它转了 90 度
I = Zoom In, O = Zoom Out, R = Rotate。

🎯 1.3.3 连续操作练习

☐ 按以下顺序连续操作，不碰鼠标：
  1. P 放一个电阻 → Esc
  2. R 旋转它
  3. W 在电阻两端画两条线 → Esc
  4. F 放一个电源符号到一条线上 → Esc
  5. G 放一个地符号到另一条线上 → Esc
💡 第一次用鼠标点菜单可能需要 10 秒，按快捷键只要 1 秒。练习 3 遍后，你的手就会记住。

🎯 1.3.4 记忆口诀

P 放器件 W 走线，N 标网名 F 电源
G 接地，R 旋转，I 放大 O 缩小
八个按键记心间，从此不点工具栏
