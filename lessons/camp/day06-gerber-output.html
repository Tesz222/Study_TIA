任务 6.1：DRC 最终检查

▶ 6.1.1 运行 DRC（设计规则检查）

DRC 会检查：线距是否足够、有没有未连接的引脚、有没有漏走的线。在投板前必须确保 DRC = 0。
☐ Display → Status
☐ 在弹出的 Status 窗口中，找到 DRC Errors：
  • 如果显示 0 → 太好了！跳到 6.2
  • 如果显示 > 0 → 需要修复

▶ 6.1.2 查找和修复 DRC 错误

☐ 按 Shift+F5（Highlight）
☐ 在 Options 面板中 → Find By Name → 选择 DRC Error
☐ 点击 OK → 所有的 DRC 错误会被高亮显示
☐ 按 F5 放大靠近一个 DRC 标记
☐ 常见 DRC 错误及修复：
标记类型含义修复方法
红色圆圈或菱形间距冲突（Line to Line spacing）把离太近的线移开
黄色标记未走完的网络（Unrouted Net）按 I 键继续走完
白色标记未连接的引脚（Unconnected Pin）检查这个引脚是否应该连接
☐ 修复一个错误后，检查 Status 中 DRC Errors 是否减少
☐ 重复直到 DRC Errors = 0

⚠ 6.1.3 特殊注意：铺铜后的 DRC

如果你铺了铜皮，Allegro 的 DRC 可能会报铜皮相关的错误（如铜皮间距不足）。常见修复：
  • 铜皮与焊盘/走线的间距不足 → 在约束管理器中调整 Shape to Line 间距
  • 铜皮有孤岛（Island）→ 不修也可以，孤岛不影响电路
☐ 确认 DRC Errors = 0

任务 6.2：丝印调整

▶ 6.2.1 生成丝印

丝印（Silkscreen）是 PCB 上的白色文字，标注位号（R1、C2、U1）和器件轮廓。板厂会根据丝印层知道器件怎么焊。
☐ Manufacture → Silkscreen

☐ 在 Silkscreen 窗口中，用默认设置 → OK
☐ 丝印会自动生成在 Board Geometry/Silkscreen_Top 层

▶ 6.2.2 检查并调整丝印位置

☐ 按 F5 放大查看丝印文字
☐ 如果丝印文字压在焊盘上或重叠在一起，需要调整：
  • 按 M 键 → 点击丝印文字（Ref Des）→ 移到空白位置 → 左键放下
  • 按 R 可以旋转丝印方向
☐ 把所有丝印文字都调整到清晰可见的位置
💡 好的丝印规范：文字不压在焊盘上、文字方向尽量统一（从左到右或从下到上）。

任务 6.3：生成 Gerber 和钻孔文件

▶ 6.3.1 配置 Gerber 输出

Gerber（RS-274X）是 PCB 制造的标准文件格式。Allegro 把 PCB 的每一层（走线层、阻焊层、丝印层）分别输出成 .art 文件。
☐ Manufacture → Artwork

☐ 在 Artwork Control Form 窗口中：
  • General Parameters 标签：保持默认（Format = RS-274X）
  • 点击 Film Control 标签

▶ 6.3.2 添加 Film（每一层对应一个 Film）

☐ 点击 Add 按钮 → 输入 TOP → OK
☐ 点击 TOP → 在右侧勾选以下 subclass：
  • 勾选 ETCH/TOP（顶层走线）
  • 勾选 PIN/TOP（顶层焊盘）
  • 勾选 VIA/TOP（顶层过孔）
☐ 点击 Add → 输入 GND_INNER → 右侧勾选 ETCH/GND_INNER
☐ 点击 Add → 输入 PWR_INNER → 右侧勾选 ETCH/PWR_INNER
☐ 点击 Add → 输入 BOTTOM → 右侧勾选 ETCH/BOTTOM、PIN/BOTTOM、VIA/BOTTOM
☐ 点击 Add → 输入 SOLDERMASK_TOP → 勾选 SOLDERMASK/TOP
☐ 点击 Add → 输入 SOLDERMASK_BOTTOM → 勾选 SOLDERMASK/BOTTOM
☐ 点击 Add → 输入 SILKSCREEN_TOP → 勾选 BOARD GEOMETRY/SILKSCREEN_TOP

▶ 6.3.3 生成 Gerber 文件

☐ 在 Film Control 中，勾选 所有 Film 前面的复选框
☐ 点击 Generate Artwork 按钮
☐ 看命令栏：应该显示 "Artwork generation completed successfully"
☐ 如果报错，通常是 Film 配置问题（某个 subclass 没勾或层名不对）

▶ 6.3.4 生成钻孔文件

钻孔文件告诉板厂在 PCB 上打多大的孔（过孔、插件引脚孔）。
☐ Manufacture → NC → Drill Parameters

☐ 保持默认设置 → OK
☐ Manufacture → NC → Drill

☐ 保持默认 → Drill 按钮
☐ 看命令栏："NC Drill completed successfully"

▶ 6.3.5 验证输出文件

☐ 打开项目文件夹 → 进入 artwork 子文件夹
☐ 确认以下文件都存在：
  • TOP.art、BOTTOM.art、GND_INNER.art、PWR_INNER.art
  • SOLDERMASK_TOP.art、SOLDERMASK_BOTTOM.art
  • SILKSCREEN_TOP.art
  • NCDrill_1.drl（钻孔文件）
☐ File → Import → Artwork → 选一个 .art 文件 → 打开查看
💡 如果你安装了 CAM350，也可以用它打开 Gerber 文件检查。很多板厂也免费提供在线 Gerber 查看器。

📋 Day 6 检查清单

  1. ☐ DRC 零错误 — Display → Status 显示 0
  2. ☐ 丝印合理 — 没有文字压在焊盘上，所有位号可见
  3. ☐ Gerber 文件生成成功 — artwork/ 目录下有 .art 文件
  4. ☐ 钻孔文件生成成功 — .drl 文件存在

常见问题

问题原因解决方法
Artwork 报 "Undefined line width for..."某层走线未定义线宽回到该层，检查走线属性，或者设置全局默认线宽
钻孔文件有 Warning孔比焊盘直径还小检查焊盘设计，通常孔直径 = 焊盘直径 / 2
Gerber 生成后看不到走线Film 配置中没勾选对应的 subclass重新检查每个 Film 勾选的 subclass 是否正确
