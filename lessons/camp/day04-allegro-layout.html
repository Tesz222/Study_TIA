任务 4.1：创建 Allegro 项目 + 叠层配置

▶ 4.1.1 启动 Allegro PCB Designer

☐ 开始菜单 → 搜索 Allegro PCB Designer → 打开
☐ Product Choices → 选 Allegro PCB Designer → Options 全勾 → Use as default → OK

▶ 4.1.2 新建 Board 文件

☐ File → New

☐ Drawing Type：Board
☐ Drawing Name：DAY4_BOARD.brd
☐ 选择一个目录保存（建议和 Day 3 的 Project 放一起）
💡 .brd 是 Allegro 的 PCB 文件格式。在 Allegro 中不存在"项目文件"的概念，只有一个 .brd 文件。

▶ 4.1.3 配置叠层（Cross-section）

叠层决定了 PCB 有多少层铜、每层多厚、绝缘层多厚。我们配一个标准的 4 层板。
☐ Setup → Cross-section
☐ 在 Cross-section Editor 中，删除不需要的层，只保留：
  • TOP — Type: Conductor（信号层）
  • GND_INNER — Type: Plane（地层，完整地平面）
  • PWR_INNER — Type: Plane（电源层）
  • BOTTOM — Type: Conductor（信号层）
☐ 选中 GND_INNER 层 → 在右侧的 Net 字段中填入 GND（给地层分配网络）
☐ 点击 OK 关闭 Cross-section Editor
💡 如果你的设计中只有 2 层板，可以不配 4 层，只保留 TOP + BOTTOM 就行。

任务 4.2：画板框 + 导入网表

▶ 4.2.1 画板框（Board Outline）

板框就是 PCB 的物理外形。Allegro 中必须先在 Board Geometry/Outline 层画一个闭合的矩形，Allegro 才知道板子有多大。
☐ Add → Line
☐ 在右侧 Options 面板中设置：
  • Class：Board Geometry
  • Subclass：Outline
  • Line Width：0（使用默认最小宽度）
  • Active Class/Subclass：同上
☐ 在绘图区内点击左键开始画框：
  • 点第一点 → 往右移 → 点第二点 → 往下移 → 点第三点 → 往左移 → 点第四点
  • 最后回到起点，右键 → Done（或按 F2）
💡 推荐尺寸：50 x 40 mm（一个不大的练习板尺寸）。如果你画的不是闭合的，Allegro 会报错。
☐ 按 F7（Zoom Fit）全屏查看完整的板框

▶ 4.2.2 设置封装库路径（重要！）

⚠️ 很多人卡在这一步。Allegro 需要知道你的封装文件放在哪里，通过 padpath 和 psmpath 来指定路径。
☐ Setup → User Preferences

☐ 在 Search Paths → Design_paths 中找到：
  • padpath：焊盘库路径，例如 C:\Cadence\SPB_17.4\share\pcb\pcb_lib\symbols
  • psmpath：封装库路径，同上
☐ 如果不知道你的 Cadence 安装路径，可以在 Windows 搜索 .dra 文件，看它们在哪
☐ 也可以新建自己的封装库文件夹：比如 D:\Cadence_Library\symbols，把路径加进去
💡 记下你设置的路径，以后所有项目都指向同一个库文件夹。

▶ 4.2.3 导入网表

☐ File → Import → Logic

☐ 在 Import Logic 窗口中：
  • 选择 Cadence 标签页
  • Import Type：DesignNetlist
  • Import directory：点击文件夹图标 → 选择 Day 3 生成的 netlistallegro 文件夹
  • 勾选 Allow etch removal when converting to FREE
  • 点击 Import Cadence
☐ 看下面的命令栏（Command Window）：
  • ✅ 如果看到 "Completed successfully" → 导入成功！
  • ❌ 如果看到 "Footprint not found: R0603" → 封装库路径没配好，或者 OrCAD 中填的封装名不对
☐ 导入成功后，按 F7 全屏，你应该看到所有器件散落在板框附近

⚠ 4.2.4 网表导入失败的解决方法

如果报"Footprint not found"，不是你的错——是封装名不匹配。解决办法：
  • 方法一：去 OrCAD 中修改 PCB Footprint 名称，重新生成网表
  • 方法二：在 Allegro 中跳过不存在的封装（不推荐，因为你需要自己去补）
  • 方法三：如果你的 Cadence 没有标准库封装，需要先创建焊盘和封装（这是另一个大的主题，Day 7 之后可以学）
💡 对于今天的练习：如果 R0603、C0402 报错，可以先在 OrCAD 中临时改为 SM0805（0805 封装的标准库名），重新生成网表再导入。

任务 4.3：摆放器件

▶ 4.3.1 快速摆放所有器件

☐ Place → Manually

☐ 在 Placement 对话框中，点击 Quick 标签
☐ 点击 Place All → 所有器件会自动放置到板框内
☐ 关闭 Placement 对话框

▶ 4.3.2 手动调整位置

☐ 按 M（Move）→ 点击一个器件 → 移动鼠标到合适位置 → 左键放下
☐ 按 R（Rotate）→ 点击器件 → 每点一次转 90 度
☐ 按下面的摆放顺序原则来布局：
  1. 先放连接器 — 固定在板框边缘（通常左边或右边）
  2. 再放核心 IC — OPA381 放在板子中央附近
  3. 最后放外围 R/C — 电阻电容紧靠对应的 IC 引脚
💡 摆放时注意飞线（Ratsnest）的变化——飞线越短说明布局越合理。飞线是黄色细线，表示两个焊盘之间需要走线。

▶ 4.3.3 检查布局

☐ 按 F7 全屏看整体效果
☐ 按 F5/F6 放大缩小检查细节
☐ 检查：是否有器件叠在一起？是否所有器件都在板框内部？
☐ 按 Ctrl+S 保存
