任务 7.1：Cross-probe 交互定位

Cross-probe 是 Cadence 最强大的功能之一——在 OrCAD 原理图上点选一个器件，Allegro PCB 上自动高亮并定位到对应的器件。反过来也同理。

▶ 7.1.1 准备工作

☐ 打开 OrCAD 中的原理图文件（Day 3 做的 .dsn）
☐ 打开 Allegro 中的 PCB 文件（Day 6 做的 .brd）
☐ 两个窗口并排显示（左右各一半）

▶ 7.1.2 从 OrCAD 定位到 Allegro

☐ 在 OrCAD 中选中一个器件（点击运放 U1）
☐ 按 Alt+Ctrl+F11（或点击工具栏的 Cross-probe 图标——一个带双向箭头的大头针）
☐ 看 Allegro — 它会自动将 U1 移动到屏幕中央并高亮
☐ 也可以在 OrCAD 中框选多个器件 → Cross-probe → Allegro 同时高亮这组器件

▶ 7.1.3 从 Allegro 定位到 OrCAD

☐ 在 Allegro 中选中一个器件（点击它）
☐ Tools → Cross-probe（或点击相同的 Cross-probe 图标）
☐ 看 OrCAD — 自动定位到对应器件
☐ 在 Allegro 中按 Shift+F5 高亮一个网络 → 再 Cross-probe → OrCAD 中对应的网络被高亮
💡 在检查 PCB 走线时非常有用：在 Allegro 中看到一根走线，想知道它在原理图上是哪个信号，高亮 → Cross-probe 即可。

🎯 任务 7.2：盲走挑战（毕业考试）

这是最后也是最关键的任务。不翻教程、不查笔记，从零开始完成以下电路设计。

考试要求：

  • 全程用快捷键，尽量不使用鼠标点击菜单
  • 遇到卡住的地方，先凭记忆解决，实在想不起来可以看一眼教程再继续
  • 计时，目标 45 分钟内完成

考试电路：单通道 TIA 前放

BOM：
  PIN 光电二极管 x1（2 pins，用 DIODE 符号代替）
  OPA381 x1（SOIC-8）
  100k 电阻 x1（R0603）
  0.68pF 电容 x1（C0402）
  0.1uF 电容 x2（C0402）
  10uF 电容 x1（C0805）
  连接器 x1（HEADER 2，2 pins）

✅ OrCAD 部分（20 分钟内）

☐ 新建 Project → Analog or Mixed A/D
☐ 按 P 放置所有器件（OPA381、DIODE、R、C、HEADER）
☐ 按 W 走线连接
☐ 按 F/G 加电源和地
☐ 按 N 给输入输出标网络名
☐ 修改 PCB Footprint（Ctrl+E）
☐ 跑 DRC（Tools → Design Rules Check）→ 0 Error
☐ 生成网表（Tools → Create Netlist）

✅ Allegro 部分（25 分钟内）

☐ 新 Board → Cross-section 配 4 层板
☐ 画板框 30x20 mm
☐ 导入网表（File → Import → Logic）
☐ 按 M / R 摆放器件
☐ 按 I 走线
☐ 铺地铜（Shape → Add Rect → GND）
☐ DRC → 0
☐ 出 Gerber + 钻孔

🏆 完成以上所有步骤 → 你毕业了！
