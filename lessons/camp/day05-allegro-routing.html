任务 5.1：手工走线

Allegro 中走线用的快捷键是 I（Add Connect）。记住 I = "I connect"。

▶ 5.1.1 设置约束规则（走线宽度）

在走线之前最好先设置好线宽规则，否则 Allegro 会用默认宽度（可能太细），走完再改就麻烦了。
☐ Setup → Constraints → Electrical Constraints（在 17.4 中可能是 Constraint Manager）
☐ 如果是 Constraint Manager：
  • 在左侧导航中选择 Physical → Net → All Layers
  • 右键 → Create → Physical CSet → 命名为 ANALOG
  • 将 Min Line Width 设为 0.3 mm（模拟信号走线）
  • Max Line Width 设为 0.5 mm
☐ 在 Net 列表中，找到你的模拟信号网络（如 SIG_IN、SIG_OUT）
☐ 在 Referenced Physical CSet 列中，选择刚创建的 ANALOG
💡 如果觉得设置约束太复杂，可以先跳过——默认规则也能走线，只是线宽是默认值。

🎯 5.1.2 开始走线（最重要的操作）

☐ 按 I（Add Connect）→ 光标变成十字
☐ 把鼠标移到一个焊盘上 → 焊盘上出现白色小方块后 → 点击左键
☐ 移动鼠标 → 一根线从焊盘延伸出来 → 到目标焊盘 → 再次左键
☐ 如果走歪了 → 按 Backspace 取消上一步
☐ 如果线走到一半需要换层 → 按 V 打个过孔 → 继续走
☐ 走完后按 F2（Done）
💡 可以先在空白区域练习走线、拐弯、Backspace 回退——不需要一开始就对着焊盘连。

▶ 5.1.3 走线顺序

建议的走线顺序（重要信号先走）：
  1. 模拟信号线（运放输入→输出）— 最关键的线，先走
  2. 电源线（VCC 到去耦电容到运放引脚）— 电源线要走宽些
  3. 地线（后面会用铜皮覆铜，可以不单独走地线）
  4. 剩下的其他信号
💡 对 TIA 电路：运放输入端的走线要尽量短（< 3 mm），反馈电阻 Rf 和 Cf 跨接在输入和输出之间。

▶ 5.1.4 走线实用技巧

操作按键说明
开始走线IAdd Connect
完成走线F2Done
取消本次走线F3Cancel
回退上一步Backspace不退出走线模式，回到上一个拐点
打孔换层VAdd Via
锁定方向按住 Shift强制走水平或垂直
切换走线方式走线时右键Toggle：45 度角 / 直角 / 弧形走线
删除一段线Delete再点击要走删除的线
☐ 练习以上每个操作至少 1 次

任务 5.2：覆铜

▶ 5.2.1 铺地铜

覆铜（Copper Pour）就是把 PCB 上空白的区域填上铜皮，通常是 GND 网络。这样所有地焊盘通过铜皮连接，不用单独走地线。
☐ Shape → Add Rect

☐ 在右侧 Options 面板中设置：
  • Class：Etch
  • Subclass：TOP
  • Assign net：GND
  • Shape type：Static Solid（静态实心铜皮）
☐ 在板框内画一个矩形覆盖整个板面 → 左键拉出矩形 → 右键 → Done
☐ 你会看到铜皮自动铺满整个板框，地焊盘上出现"X"形连接标记

▶ 5.2.2 挖空敏感区域

在 TIA 运放的反相输入焊盘下方，我们需要挖掉铜皮——因为铜皮和焊盘之间的寄生电容会影响电路稳定性（Lesson 0004 讲过）。
☐ Shape → Void → Circular

☐ 在运放输入焊盘周围画一个圆圈 → 左键点圆心 → 拖出半径 → 左键确定
☐ 铜皮上出现了一个圆洞，焊盘和铜皮之间有了间隙
💡 如果铺的是动态铜皮（Dynamic Shape），可以用 Shape → Void → Auto 自动避让。

任务 5.3：进阶操练

▶ 5.3.1 高亮网络检查走线

☐ 按 Shift+F5（Highlight）
☐ 在右侧 Options 面板中 → Find By Name → Net
☐ 输入你要高亮的网络名（如 SIG_IN）→ OK
☐ 那个网络的所有走线会变成高亮颜色，检查路径是否合理
☐ 按 Shift+F6（Dehighlight）取消高亮

🎯 5.3.2 快捷键强化练习

下面这个序列要练到不用想就能按出来：
  1. I → 走一小段线 → V → 打个孔 → 继续走 → F2
  2. M → 移一下器件 → R → 旋转 → F2
  3. Delete → 删除一段走歪的线 → F2
  4. Shift+F5 → 高亮 → Shift+F6 → 取消
☐ 每个序列重复 3 遍

  3. ☐ 会用 V 打孔 — 走线中按 V 能换层
