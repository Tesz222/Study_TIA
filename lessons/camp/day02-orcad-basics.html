任务 2.1：创建项目并放置第一个元件

▶ 2.1.1 新建 Project

☐ 打开 OrCAD Capture（如果没开的话）
☐ File → New → Project
☐ 在弹出的对话框中：
  • Name：输入 DAY2_PRACTICE
  • Location：选一个你方便找的文件夹（建议新建一个 Cadence_Learning 文件夹）
  • 选中 Analog or Mixed A/D → OK
💡 Analog or Mixed A/D 这个选项意味着该 Project 支持 PSpice 仿真（如果以后用到的话）。画原理图选这个没错。
☐ 确认项目管理器（左侧）中出现了 DAY2_PRACTICE.dsn（原理图文件）

▶ 2.1.2 放置一个运放（LM358）

我们先用 Cadence 标准库中的 LM358（双运放），它足够简单。
☐ 按 P（Place Part）
☐ 在弹出的 Place Part 窗口中，点击 Add Library 按钮（文件夹图标）
☐ 找到 C:\Cadence\SPB_17.4\tools\capture\library\opamp.olb 并打开
☐ 在 Part List 中搜索 LM358
☐ 双击 LM358 → 把它放到原理图页上（再点一下左键确定位置）
☐ 按 R 旋转几次，调整方向
☐ 按 Esc 退出放置模式
💡 LM358 是双运放（Parts per Package = 2），所以页面上显示两个器件：U1A 和 U1B。但我们只需要一个，删掉 U1B 或用它做另一个通道。

▶ 2.1.3 放置电阻和电容

☐ 按 P 打开 Place Part
☐ 点击 Add Library → 找到并加载 DISCRETE.olb（在同样的 library 文件夹下）
☐ 在 Part List 中找到 R（电阻）→ 双击 → 放 4 个 → Esc
☐ 重复操作，找到 C（电容）→ 放 2 个 → Esc
☐ 放完后按 I 放大检查，确认所有器件都在页面上

任务 2.2：走线连接 + 电源地 + 网标

▶ 2.2.1 布线（Wire）

先认识一下 LM358 的引脚（以 DIP-8 封装为例）：
Pin 1 = OUT_A（A 路输出）、Pin 2 = -IN_A（A 路反相输入）、Pin 3 = +IN_A（A 路同相输入）
Pin 4 = VEE（负电源，这里接 GND）、Pin 8 = VCC（正电源）
☐ 按 W（Place Wire），光标变成十字
☐ 点击 LM358 的 Pin 3（同相输入端）→ 把线引出来 → 在需要拐弯的地方点左键 → 连到 R1 的一端
☐ 继续连接以下网络（建议按这个顺序）：
  • R1 另一端 → 悬空（这是输入信号端口）
  • Pin 2 → R2 一端（反馈电阻）
  • R2 另一端 → Pin 1（输出）
  • Pin 1 → 另一个电阻 R3 → 输出端（悬空）
  • Pin 3 到 GND 之间加 R4（偏置电阻）
☐ 走线时按住 Shift 可以锁定水平/垂直方向
☐ 全部走完后按 Esc 退出走线模式
☐ 放大检查：每个连接点应该有 红色小圆点。如果没有，说明线没连上。

▶ 2.2.2 加电源和地

☐ 按 F（Place Power）
☐ 在 Library 列表中找到 SOURCE 库 → 选 VCC → 放到 LM358 的 Pin 8（VCC）旁边
☐ 按 W 将 VCC 符号连到 Pin 8
☐ 按 G（Place Ground）
☐ 在 Library 中选 SOURCE → 选 0（/GND 符号）→ 放两个：一个到 Pin 4（VEE），一个到 R4 接地端
☐ 按 W 连好接地线

▶ 2.2.3 标注网络名（Net Alias）

网络名就是给导线起名字。没有名字的线用系统自动名（N0001、N0002），有了名字才好阅读和后续检查。
☐ 按 N（Place Net Alias）
☐ 输入 SIG_IN → 点击靠近 R1 输入端的线（网络名会标在这根线上）
☐ 再次按 N → 输入 SIG_OUT → 点击 R3 输出端的线
☐ 再次按 N → 输入 +5V → 点击 VCC 符号上的线
💡 注意：网络名不要有空格，不要以数字开头，尽量用下划线代替空格。

任务 2.3：修改参数并保存

▶ 2.3.1 修改器件值（Value）

☐ 双击 R1（电阻符号上双击）
☐ 在弹出的 Property Editor 窗口中，找到 Value 属性，把默认值改为 10k
☐ 对其他 3 个电阻也做同样操作：R2=100k、R3=1k、R4=10k
☐ 双击两个电容，Value 改为 0.1uF（去耦电容）
💡 可以按住 Ctrl 选中多个器件，右键 → Edit Properties（或按 Ctrl+E），在一张表里批量修改。

▶ 2.3.2 修改 PCB Footprint

PCB Footprint 告诉 Allegro"这个器件在 PCB 上长什么样"。这一步可以不做，但为了后面的 Day 4 做准备，建议现在设好。
☐ 选中 R1 → Ctrl+E → 找到 PCB Footprint 属性 → 改为 R0603
☐ 对其他电阻和电容做同样操作（R0603 或 R0402、C0402）
☐ 选中 LM358 → PCB Footprint 改为 DIP8（通孔封装）或 SOIC8（贴片封装）

▶ 2.3.3 保存

☐ 按 Ctrl+S 保存（养成随时保存的习惯！）
