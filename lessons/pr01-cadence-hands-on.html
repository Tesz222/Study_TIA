实操：单通道 TIA 电路 — OrCAD 原理图到 Allegro 投板

Lesson PR01 · 实战演练 · 预计用时 60 分钟

实操目标 跟着步骤，在 Cadence 中完整走完一遍：从打开 OrCAD 画原理图，到 Allegro 布局布线，最后出 Gerber。每一步都告诉你按什么键、点哪里。

电路说明

我们要画的是一最简单的单通道 TIA 电路：

  • 光电二极管 PD1 → OPA381（TIA）→ 输出 VOUT
  • Rf = 100 kΩ，Cf = 0.68 pF
  • 单电源 +5 V 供电
  • BOM：PD x1, OPA381 x1, R x3, C x4, 连接器 x1, TP x3

第一部分：OrCAD Capture 原理图（预计 25 min）

Step 0：启动 OrCAD Capture — 选对 Product Choice

操作：开始菜单 → 搜索 OrCAD Capture CIS → 打开

第一次启动时，会弹出 Product Choices 窗口，这是 Cadence 让你选择以哪个产品模式运行：

  • 选：OrCAD Capture CIS（列表中找到这一项）
  • 如果你公司买的是 Allegro 套装，选 Allegro Design Entry CIS
  • 如果你接下来不需要元器件数据库（CIS），选 OrCAD Capture 启动更快
勾选左下角 Use as default，下次就不会再弹这个窗口了

Product Choices 速查

这个选项含义你什么时候选
OrCAD Capture纯原理图，无 CIS 无 PSpice个人画简单原理图，启动最快
OrCAD Capture CIS原理图 + 元器件库管理默认推荐
OrCAD PSpice Designer原理图 + 仿真需要仿真模拟电路时
Allegro Design Entry CISAllegro 版原理图公司用 Allegro 套装时
注意：如果选错了或想切换产品特性，不需要退出重开——在 OrCAD 菜单栏上 File → Change Product 可以随时切换。

Step 1：创建项目

操作：File → New → Project
  • Name：TIA_AMP_PRACTICE
  • Location：选一个你常用的工作目录
  • 选中 Analog or Mixed A/D（带 PSpice 仿真能力）
  • 点击 OK
确认项目管理器中出现了 .dsn 文件（原理图文件）和 .opj 文件（项目文件）

Step 1：放置 OPA381

操作：按 P 键打开 Place Part 面板
  • 先确认 Libraries 里有没有 OPA381。如果没有：
  • 点击 Add Library 按钮（文件夹图标）
  • 找到 Cadence 自带库路径：...\tools\capture\library\opamp.olb
  • 在 Part List 中搜索 OPA381，如果找不到就用 LM741 代替（只是练习）
  • 双击放置到原理图。按 R 可以旋转方向
  • 按 Esc 或右键 → End Mode 退出放置
确认原理图页上出现了一个运放符号，默认位号 U1

Step 2：放置电阻、电容和光电二极管

操作：按 P 继续放置
  • 在 Libraries 中添加 DISCRETE.olb（在 capture\library 下）
  • 搜索并放置：
  • R（电阻）× 3 个：R1, R2, R3
    — R1 = Rf（反馈电阻，稍后改值为 100k）
    — R2 = 10k（偏置电阻）
    — R3 = 1k（输出限流）
  • C（电容）× 3 个：C1, C2, C3
    — C1 = Cf（反馈电容，0.68 pF）
    — C2 = 0.1 uF（去耦）
    — C3 = 10 uF（去耦）
  • 搜索 LED 或 PHOTODIODE：如果没有，用一个普通二极管 D（DIODE）代替
    （练习中只是走通流程，符号不重要）
按 I 键放大查看，确认所有器件都在原理图页上

Step 3：修改器件参数

修改位号和 Value：

  1. 双击电阻 R1 弹出属性框
  2. 将 Value 改为 100k
  3. 将 PCB Footprint 改为 R0603
  4. 对 R2（10k）、R3（1k）做同样操作
  5. 对 C1 将 Value 改为 0.68pF，Footprint 改为 C0402
  6. 对 C2 改 Value 为 0.1uF
  7. 对 C3 改 Value 为 10uF

批量修改的快捷技巧：

  • 选中一个器件 → 右键 → Edit Properties（快捷键 Ctrl+E）
  • 在 Property Editor 中，可以看到所有器件列表，直接在一张表里修改所有值

Step 4：走线连接

操作：按 W 键开始走线

按以下连接方式逐个连接（从左上到右下布局）：

  PD1（光电二极管）
    ├ 阴极 → U1 Pin 2（OPA381 反相输入）
    └ 阳极 → GND

  U1（OPA381）
    Pin 2（-IN）→ R1 一端
    Pin 3（+IN）→ R2 到 GND
    Pin 4（VEE）→ GND（单电源供电）
    Pin 6（OUT）→ R1 另一端、C1 另一端、R3 一端
    Pin 7（VCC）→ +5V
    Pin 1、5、8 → 不连（NC）

  R3 另一端 → 输出接插件（CON1 / HEADER 2）

  +5V → C2 一端、C3 一端
  C2 另一端 → GND
  C3 另一端 → GND
走线时按住 Shift 可锁定水平/垂直方向

Step 5：放置电源和地

  • 按 F 键（Place Power）
    — 在 Library 中选择 SOURCE 库
    — 选择 VCC（或 +5V）符号，放置到 U1 的 Pin 7
  • 按 G 键（Place Ground）
    — 在 Library 中选择 SOURCE 库
    — 选择 0（或 GND）符号，放置到 PD 阳极、U1 Pin 3/4、C2/C3 另一端

Step 6：标注网络名

操作：按 N 键（Place Net Alias）
  • 输入 PD_IN，放在 PD 阴极到 U1 Pin 2 的线上
  • 输入 TIA_OUT，放在 U1 Pin 6 的输出线上
  • 输入 +5V_A，放在电源线上（区分模拟电源）
走线完成后，按 Ctrl+S 保存，确保所有连接点出现红色的小圆点

Step 7：Annotate（自动编号）

如果之前手动放了器件，位号可能不连续或有冲突：

Tools → Annotate
  • 勾选 Reset part references to "?"（不勾也可以，直接选 Incremental）
  • 选中 Incremental reference update
  • 点击 OK

系统会自动将位号整理为 R1、R2、R3、C1、C2、C3、U1、D1 等

Step 8：DRC 检查

操作：Tools → Design Rules Check
  • 勾选 Check single node nets（检查孤立节点）
  • 点击 OK
  • 查看 Session Log 下方的信息。如果有错误（红色 X），双击错误条目定位到原理图上修改
  • 常见错误：Floating input pin（输入引脚没连上）
确认 DRC 结果为 0 Errors（可以有 Warnings）

Step 9：生成网表

操作：Tools → Create Netlist
  • 选择 PCB Editor 标签页
  • 勾选 Create PCB Editor Netlist
  • 确定 PCB Footprint 也勾选了
  • 点击 确定
确认 Session Log 显示 "Netlist successfully completed"
打开项目文件夹，看是否有 netlistallegro 文件夹生成

第二部分：Allegro PCB Editor 布局布线（预计 30 min）

Step 10：启动 Allegro PCB Editor — 选对 Product Choice

操作：开始菜单 → 搜索 Allegro PCB Designer → 打开

同样会弹出 Product Choices 窗口，这次你要选的是 PCB 布局相关选项：

  • 选：OrCAD PCB Designer Professional（如果你用的 OrCAD 套装，这是默认推荐）
  • 选：Allegro PCB Designer（如果你用的 Allegro 套装）
  • 不确定？先问同事公司买的哪个版本。问不到就选 OrCAD PCB Designer Professional
同样勾选 Use as default，下次不弹窗

💡 如果 Product Choices 窗口中有 Options 列表，建议把能勾的全勾上，避免后续用到某个功能时提示 License 不足。

建新文件：File → New
  • Drawing Type：Board
  • Name：TIA_AMP_BOARD.brd
  • Template 可以选 Allegro 自带的 board_template_4layer.template 或先不选，后面手动设

Step 11：设置叠层（简单版 — 2 层板）

操作：Setup → Cross-section
  • 删除多余的层，保留：
  • TOP — 信号层，做模拟走线
    BOTTOM — 信号层，辅助走线 + 地铜填充
  • 在 TOP 下添加一个平面层（右键 Add Layer Above）：
  • GND — Plane 类型，网络赋值为 GND
    （如果做 4 层板，可以加 TOP/GND/PWR/BOTTOM）
确认 Cross-section 编辑器中没有红色报错

Step 12：设置板框（Outline）

操作：用 Add → Line 画板框
  • 在 Options 面板中：
  • Class：Board Geometry
    Subclass：Outline
  • 画一个 30 mm x 20 mm 的矩形板框
  • 画完后在工具面板里点 Done（或按 F2）结束

Step 13：导入网表

操作：File → Import → Logic
  • 选择 Cadence 标签页
  • Import Type：DesignNetlist
  • Import directory：点击右边的文件夹图标，选择刚才 OrCAD 生成的 netlistallegro 文件夹
  • 确保勾选 Allow etch removal when converting to FREE
  • 点击 Import Cadence
如果提示找不到封装（Footprint not found），说明 OrCAD 中对某个器件的 PCB Footprint 名称和 Allegro 中的 .dra 文件名不一致。记下缺失的封装名，后面需要创建或映射。
导入成功后，按 F6 缩小，你应该看到器件散落在板框附近

Step 14：放置器件（布局）

操作：Place → Manually
  • 在弹出的 Placement 对话框中：
  • 选中 Quick 标签，点击 Place All（所有器件先自动放置到板框内）
  • 关闭对话框

手工精细布局：

  • 按 M（Move），点击一个器件开始移动
  • 按 R 旋转
  • 按 F5 放大 / F6 缩小

推荐的布局位置（左到右）：

  [PD1] → [R1/C1] → [U1 OPA381] → [R3] → [CON1]
                           ^
                          [R2]
                           v
                         [C2/C3 去耦电容]
  • PD1 紧靠 U1 的 Pin 2（反相输入），间距 ≤ 3 mm
  • R1（反馈电阻）和 C1（反馈电容）跨在 U1 的 Pin 2 和 Pin 6 之间
  • R2 在 U1 的 Pin 3 到 GND 之间
  • C2（0.1 uF）和 C3（10 uF）紧靠 U1 的 Pin 7（VCC）
  • CON1 在板框最右边
布局完成后，按 F7 缩小看全板效果

Step 15：设置约束规则

操作：Setup → Constraints → Constraint Manager
  • 在 Physical 标签页中：
  • 右键 → Create → Physical CSet
    — 名称：ANALOG
    — Min Line Width：0.2 mm
    — Default Via：选择一个你创建好的过孔（如果没有，先创建：Setup → Via → Define）
  • 在 Spacing 标签页中：
  • 右键 → Create → Spacing CSet
    — 名称：ANALOG_SPACE
    — Line to Line：0.2 mm
  • 回到 Constraint Manager 主界面：
  • 在 Net → Physical 中，找到 PD_IN 和 TIA_OUT 网络
    在 Referenced Physical CSet 列中，选择 ANALOG

Step 16：手动走线

操作：按 I 键开始走线（Add Connect）

走线顺序（先走最重要的模拟信号）：

  1. PD1 到 U1 Pin 2（最关键——TIA 输入）
    — 最短路径 → 直接连，不要绕弯
    — 走线宽度 0.3 mm
  2. U1 Pin 6 到 R3 到 CON1（TIA 输出）
    — 和输入保持距离
  3. R1 和 C1（反馈网络）
    — 跨接在 Pin 2 和 Pin 6 之间
  4. R2（同相端偏置）
    — Pin 3 到 GND
  5. 电源走线
    — Pin 7 先到 C2 → C3 → 然后走宽线（0.5 mm）到连接器
  6. GND 走线
    — 尽量使用星形接地（所有 GND 点汇聚到一点）
走线过程中按 V 键打孔换层；按 Backspace 取消上一步
走线完成后按 F2（Done），检查是否有未连接的线（显示的飞线）

Step 17：覆铜（静态铜皮）

操作：Shape → Add Rect（快捷键要自己配置，或直接点菜单）
  • 在 Options 面板中：
  • Class：Etch
    Subclass：TOP
    Assign net：GND
    Shape type：Static Solid
  • 在板框内画一个矩形覆盖整个板面
  • 铺好后，如果有不需要铺铜的地方，用 Shape → Void → Circular 挖掉
  • 在 PD_IN 焊盘周围画一个圈挖掉地铜（减少寄生电容！）
覆铜后检查走线和铜皮间距是否满足约束规则

Step 18：DRC 最终检查

操作：Display → Status
  • 查看 DRC Errors 是否为 0
  • 如果有错误：
  • 按 Shift+F5（Highlight），把 DRC 错误高亮出来
    按 F5 放大查看错误位置
    修改后按 Shift+F6 取消高亮
确认 Unconnected Pins = 0
Unrouted Nets = 0
DRC Errors = 0

Step 19：出 Gerber（投板文件）

操作：Manufacture → Artwork
  • 在 Film Control 中配置：
  • 点击 Add 按钮，添加以下 Film：
    — TOP（顶层走线）
    — BOTTOM（底层走线）
    — SOLDERMASK_TOP（顶层阻焊）
    — SOLDERMASK_BOTTOM（底层阻焊）
    — SILKSCREEN_TOP（顶层丝印）
    — NC_DRILL（钻孔）
  • 勾选所有 Film，点击 Generate Artwork
  • 再生成钻孔文件：Manufacture → NC → Drill
  • 点击 Drill 按钮
确认所有 Gerber 文件生成到 .\artwork\ 目录下
用 CAM350 或 Allegro 自带的 ViewPlot 打开其中一个 .art 文件检查

完整流程图（一步接一步）

OrCAD Capture:
  [0] New Project (.opj)
  [1] Place OPA381   (P)
  [2] Place R/C/PD   (P)
  [3] Edit Values    (双击)
  [4] Wire           (W)
  [5] Power & GND    (F / G)
  [6] Net Alias      (N)
  [7] Annotate       (Tools)
  [8] DRC            (Tools)
  [9] Netlist        (Tools)
         │
         ▼
Allegro PCB:
  [10] New Board (.brd)
  [11] Cross-section  (Setup)
  [12] Board Outline  (Add/Line)
  [13] Import Netlist (File/Import/Logic)
  [14] Place Parts    (Place/Manually)
  [15] Manual Layout  (M / R)
  [16] Constraints    (Setup/Constraints)
  [17] Route          (I)
  [18] Copper Pour    (Shape/Add Rect)
  [19] DRC            (Display/Status)
  [20] Gerber Out     (Manufacture/Artwork)
  [21] Drill Out      (Manufacture/NC/Drill)
         │
         ▼
  投板！
做完之后

如果这是你第一次完整走完 Cadence 流程，恭喜你！你已经掌握了从原理图到投板的完整能力。

建议每个步骤多做几次，目标是：
- 不看这个页面能走完一遍
- 知道每个快捷键按什么
- 知道遇到错误去哪里查（Session Log / DRC / 封装库路径）
