Cadence OrCAD + Allegro 七天入门到熟练

完整自学路线 · 每天 2-3 小时 · 从零到能画板投板

学习目标
第 1-2 天：掌握 OrCAD Capture 原理图绘制，能画简单电路
第 3 天：掌握原理图进阶（建库、DRC、网表）
第 4-5 天：掌握 Allegro PCB Editor 布局布线，能画 2 层板
第 6 天：掌握 Gerber 输出和设计检查
第 7 天：综合实战，完整体验原理图 → PCB → Gerber 全流程

📅 第 1 天：环境搭建与界面导航

耗时：2 小时  |  核心键：P W F G R I O

1.1 启动 Cadence（20 min）

  • 开始菜单找到 Capture CIS 和 PCB Editor，发送快捷方式到桌面
  • 打开 Capture CIS，第一次启动时 Product Choices 选 OrCAD Capture CIS，勾选 Use as default
  • 认识界面四个区域：项目管理器、原理图编辑区、工具栏、Session Log
  • File → Change Product 随时切换产品特性
✅ 能独立启动 Capture CIS，知道 Product Choices 的含义

1.2 创建第一个项目（20 min）

  • File → New → Project，选 Analog or Mixed A/D
  • 项目名：TIA_PRACTICE
  • 认识生成的文件：.opj（项目文件）、.dsn（原理图文件）、.olb（库文件）

1.3 放置第一个器件（30 min）

  • 按 P 打开 Place Part 面板
  • 添加库：...\tools\capture\library\opamp.olb 和 discrete.olb
  • 搜索 LM741 放置一个运放
  • 搜索 R 放置一个电阻
  • 按 R 旋转，按 I/O 放大缩小

1.4 走线和放置电源地（30 min）

  • 按 W 在两个器件之间走线
  • 按 F 放置电源符号（选 VCC）
  • 按 G 放置地符号（选 0/GND）
  • 按 Esc 退出当前操作
  • 按 Ctrl+S 保存

1.5 今日复习（20 min）

今日快捷键：P 放器件   W 走线   F 电源   G 地   R 旋转   I 放大   O 缩小   Ctrl+S 保存   Esc 退出
打开 Capture，新建空白页，练习 5 遍：P → 放电阻 → R 旋转 → W 走线 → F 电源 → G 地 → Ctrl+S → Esc

📅 第 2 天：原理图设计入门

耗时：2.5 小时  |  核心键：N T E Tab

2.1 认识库文件结构（20 min）

  • Cadence 标准库路径：...\tools\capture\library\
  • 常用库：discrete.olb（R, C, L, D, Q）, opamp.olb（运放）, connector.olb（连接器）
  • 区分：库文件 (.olb) vs 设计文件 (.dsn) vs 项目文件 (.opj)

2.2 完整原理图：单级 TIA（60 min）

  • 放置器件：1x OPA381 + 1x 光电二极管 + 3x 电阻 + 3x 电容 + 1x 连接器
  • 按 P 放器件，按 Tab 连续放置多个
  • 按 W 走线连接
  • 按 F/G 加电源/地
  • 按 N 添加网络别名 PD_IN、TIA_OUT
  • 按 T 添加文字注释
  • 双击器件修改 Value（R1=100k, C1=0.68pF 等）
  • 按 Ctrl+E 批量修改属性

2.3 设计规则与检查（30 min）

  • Annotate：Tools → Annotate → Incremental reference update → OK（自动重排位号）
  • DRC：Tools → Design Rules Check → 勾选 Check single node nets → OK
  • 查看 Session Log，双击错误条目定位修改
  • 保存：Ctrl+S

2.4 今日复习（20 min）

新增快捷键：N 网络别名   T 文字   Ctrl+E 编辑属性   Tab 连续放置   Ctrl+Z 撤销
打开 .dsn → P 放器件 → W 走完 → N 标网名 → Tools → DRC → 检查通过 → Ctrl+S 保存。今天的目标：完整画出 TIA 原理图且 DRC 零错误。

📅 第 3 天：原理图进阶——建库与网表

耗时：2.5 小时  |  核心概念：.olb 创建、PCB Footprint、Netlist

3.1 创建自定义器件符号（45 min）

  • File → New → Library，创建 my_lib.olb
  • 在 .olb 上右键 → New Part
  • 填写参数：Name=OPA381, PCB Footprint=SOIC8, Parts per Package=1
  • 用 Place Pin 放置 8 个引脚：
  • Pin 1 (NC)   Pin 2 (-IN, Input)   Pin 3 (+IN, Input)
    Pin 4 (VEE, Power)   Pin 5 (NC)   Pin 6 (OUT, Output)
    Pin 7 (VCC, Power)   Pin 8 (NC)
  • 保存后按 P 刷新即可在库中找到
✅ 能独立创建一个标准 IC 的原理图符号

3.2 理解 PCB Footprint 的映射（30 min）

  • 原理图中每个器件必须有 PCB Footprint 属性，值必须和 Allegro 中的封装文件名一致
  • 常见封装命名：R0603、C0402、SOIC8、TSSOP28
  • 在 Property Editor 中批量设置 Footprint
  • Tools → Create Netlist → PCB Editor 标签页 → 勾选 PCB Footprint 检查

3.3 生成网表并检查错误（30 min）

  • Tools → Create Netlist → PCB Editor 标签页 → Create PCB Editor Netlist → 确定
  • 检查 Session Log 是否显示 "Netlist successfully completed"
  • 常见错误及解决：
  • - Footprint not found：封装名不匹配，去 Allegro 中查正确的 .dra 文件名
    - Pin number mismatch：原理图引脚编号和封装焊盘编号不一致
    - Net syntax error：网络名包含了非法字符（空格、特殊符号）
  • 生成的文件在项目目录下的 netlistallegro 文件夹中

3.4 今日复习（20 min）

今日核心：New Part 对话框的每一项含义  |  PCB Footprint 是 OrCAD 和 Allegro 之间的"桥梁"  |  网表错误 80% 来自 Footprint 名称不匹配
创建一个 OPA381 符号 → 分配 Footprint → 生成网表零错误。如果你的网表报错，先检查 Footprint 名称。

📅 第 4 天：Allegro PCB Editor 入门

耗时：3 小时  |  核心键：M I R V F5 F6 F7

4.1 认识 Allegro 界面（20 min）

  • 开始菜单 → Allegro PCB Designer → Product Choices 选 Allegro PCB Designer（Options 全勾上）
  • 界面五大区域：菜单栏、Options 面板（右键/双击弹出）、Find 面板（过滤选择对象）、工作区、命令行（底部输入命令）
  • 认识 Class/Subclass 体系：Etch（走线层）、Board Geometry（板框）、Package Geometry（封装外形）、Silkscreen（丝印）

4.2 叠层与板框（30 min）

  • Setup → Cross-section：配置叠层
  • 2 层板：TOP（走线） + BOTTOM（走线+地铜）
    4 层板：TOP + GND + PWR + BOTTOM（模拟设计推荐）
  • 画板框：Add → Line，Class=Board Geometry，Subclass=Outline
  • 画一个 40x30 mm 的矩形

4.3 导入网表与布局（40 min）

  • File → Import → Logic → Cadence → Import directory 选择 netlistallegro 文件夹 → Import Cadence
  • 导入成功后按 F7 全屏查看
  • Place → Manually → Quick 标签 → Place All（全部放入板框）
  • 按 M 移动器件，按 R 旋转
  • 布局原则：
  • - PD 紧靠运放输入 (≤ 3 mm)
    - 去耦电容紧靠 IC 电源引脚 (≤ 5 mm)
    - 连接器靠板框边缘

4.4 手动走线入门（40 min）

  • 按 I 进入走线模式（Add Connect）
  • 点击焊盘开始走线，点击目标焊盘结束
  • 按 V 在走线中打孔换层
  • 按 Backspace 回退上一步（对应 Allegro 的 Oops 命令）
  • 走完按 F2（Done）结束
  • 走线宽度在 Options 面板设置：模拟信号 0.3 mm，电源 0.5 mm，普通 0.25 mm

4.5 今日复习（20 min）

今日快捷键：M 移动   I 走线   R 旋转   V 打孔   F2 完成   F5/F6 放大缩小   F7 全屏   Backspace 回退
今天的目标：把原理图成功导入 Allegro，器件摆好，至少走通一个信号。第一次看到飞线消失的感觉很棒。

📅 第 5 天：PCB 布局进阶

耗时：3 小时  |  核心功能：约束管理器、覆铜、Room 布局

5.1 约束管理器（30 min）

  • Setup → Electrical Constraints（或 Setup → Constraints → Constraint Manager）
  • Physical 标签：右键 → Create → Physical CSet → 名称 ANALOG
  • Min Line Width = 0.2 mm，Max = 0.5 mm
  • Spacing 标签：右键 → Create → Spacing CSet → 名称 ANALOG_SPACE
  • Line to Line = 0.2 mm
  • 将模拟信号网络（PD_IN、TIA_OUT）关联到 ANALOG 约束集

5.2 完整布线（60 min）

  • 走线顺序：先模拟信号 → 再电源 → 最后普通信号
  • 第一优先级：PD_IN（TIA 输入，走线最短，包地）
  • 第二优先级：TIA_OUT（输出，远离输入）
  • 第三优先级：电源（宽线 0.5 mm）
  • 第四优先级：其他信号
  • 走线技巧：按 Shift 锁定水平/垂直

5.3 覆铜（30 min）

  • Shape → Add Rect（或 Add Polygon）
  • Options：Class=Etch，Subclass=TOP，Assign net=GND
  • 画一个矩形覆盖板框内部的空白区域
  • 在 PD_IN 焊盘下方：Shape → Void → Circular → 画圈挖掉铜皮（减少寄生电容）
  • 底层同样操作（Subclass=BOTTOM）

5.4 模拟地 vs 数字地处理（30 min）

  • 如果板上有模拟和数字部分，建议用 0 Ω 电阻或磁珠做单点连接
  • 在覆铜时不直接连接，铺两块独立的 GND 铜皮
  • 用一条走线（或 0 Ω 电阻）连接模拟地和数字地

5.5 今日复习（20 min）

今日核心命令：Setup → Electrical Constraints   Shape → Add Rect 覆铜   Shape → Void 挖空
今天的目标：走完所有主要信号、覆铜完成、DRC 错误 ≤ 5 个。如果走线遇到瓶颈，先走完关键的模拟信号，其他信号可以放宽。

📅 第 6 天：DRC、Gerber 输出与设计检查

耗时：2.5 小时  |  核心技能：DRC 检查、Gerber 配置、钻孔文件

6.1 DRC 错误排查（40 min）

  • Display → Status 查看 DRC 错误总数
  • 按 Shift+F5 高亮错误：在 Options 面板中选择 DRC Errors
  • 常见错误及修正：
  • - Line to Line Spacing：两根走线间距不足 → 拉开或调整约束
    - Unconnected Pin：有引脚没走线 → 连接或确认 NC
    - Unrouted Net：网络未完成布线 → 走完或检查是否被忽略
    - Stub：走线分支 → 对模拟信号可以不处理，右键 Mark as satisfied
  • 按 Shift+F6 取消高亮

6.2 Gerber 文件输出（50 min）

  • Manufacture → Artwork
  • Film Configuration：点击 Add 添加以下 Film
  • - TOP：Etch/TOP + Pin/TOP + Via/TOP
    - BOTTOM：Etch/BOTTOM + Pin/BOTTOM + Via/BOTTOM
    - SOLDERMASK_TOP：Soldermask/TOP
    - SOLDERMASK_BOTTOM：Soldermask/BOTTOM
    - SILKSCREEN_TOP：Silkscreen/TOP + Package Geometry/Silkscreen_Top
    - NC_DRILL：钻孔图
  • 点击 Generate Artwork，生成 .art 文件到 .\artwork\ 目录

6.3 钻孔文件（20 min）

  • Manufacture → NC → NC Drill Parameters → 设置格式（可以使用默认值）
  • Manufacture → NC → NC Drill → Drill → 生成 .drl 文件
  • Manufacture → NC → NC Route（如果板上有槽孔）

6.4 投板前的检查清单（20 min）

投板检查清单
✅ DRC 零错误（Display → Status）
✅ 所有网络已连接（Unrouted Nets = 0）
✅ 所有引脚已连接（Unconnected Pins = 0）
✅ Gerber 已生成（artwork 文件夹中有 .art 文件）
✅ 钻孔文件已生成（.drl 文件）
✅ 用 CAM350 打开 Gerber 检查无误
✅ 板框尺寸正确（Setup → Drawing Size）
✅ 丝印清晰、无重叠
✅ 所有器件封装方向一致
✅ 关键模拟信号有包地或间距足够

6.5 今日复习（20 min）

今天的目标：DRC 零错误 + 完整 Gerber 输出。如果 DRC 错误实在消不完，先解决所有 Unconnected Pins 和 Unrouted Nets，间距警告可以酌情放宽。

📅 第 7 天：综合实战 + 全流程盲操

耗时：3 小时  |  核心目标：不翻教程走完一遍完整流程

7.1 全流程盲操（120 min）

任务：从零开始设计一个双通道 TIA 电路板

  • 规格要求：
  • - 2 通道 TIA（2x OPA381）
    - 单电源 +5V
    - 输出经 2-pin 连接器引出
    - 板卡尺寸 30x20 mm
  • OrCAD 部分（< 40 min，不翻教程）
  • - 创建项目 (1 min)
    - 放置器件 (10 min)
    - 走线 + 网络别名 (15 min)
    - 电源和地 (3 min)
    - Annotate (1 min)
    - DRC (2 min)
    - 生成网表 (2 min)
    - 检查 Session Log 无错误
  • Allegro 部分（< 60 min，不翻教程）
  • - 创建 Board，设置叠层 (3 min)
    - 画板框 (3 min)
    - 导入网表 (2 min)
    - 布局 (15 min)
    - 设置约束 (3 min)
    - 走线 (20 min)
    - 覆铜 (5 min)
    - DRC 检查 (3 min)
    - 出 Gerber + 钻孔 (5 min)

7.2 常见错误速查表（20 min）

错误现象可能原因解决方法
OrCAD DRC 报 ERROR(ORCAP-3606)引脚类型冲突检查元件的 Pin Type 设置（Input/Output/Power）
网表生成失败：Footprint not foundPCB Footprint 名称不匹配打开 Allegro，查看正确的 .dra 文件名
Allegro 导入网表后无器件Import directory 指向错误确认指向 netlistallegro 文件夹
走线时提示违反约束Spacing 规则过严修改约束或调整走线
覆铜后看不到铜皮Z-order 问题或未 Assign net确认铜皮已 Assign net，按 Shift+F5 刷新
Gerber 打开后缺失一层Film 配置缺少 subclass检查 Artwork 中该 Film 包含的 Subclass
钻孔文件不对齐单位和格式设置不一致NC Drill 中的格式必须和板厂要求一致
丝印重叠看不清位号放在器件上面手动调整位置，或使用 Silkscreen → Auto Silkscreen

7.3 最终快捷键盲测（10 min）

OrCAD 盲测（8 个键）
P ?   W ?   N ?   F ?   G ?   R ?   I ?   O ?

Allegro 盲测（7 个键）
M ?   I ?   R ?   V ?   F5 ?   F6 ?   F7 ?

答案：P=放器件 W=走线 N=网络别名 F=电源 G=地 R=旋转 I=放大 O=缩小 | M=移动 I=走线 R=旋转 V=打孔 F5=放大 F6=缩小 F7=全屏

7.4 毕业总结（10 min）

7 天学完后你应该能：

✅ 独立用 OrCAD Capture 画原理图（放置、走线、网络标号、电源地、DRC、网表）
✅ 创建自定义器件符号并分配封装
✅ 用 Allegro PCB Editor 布局布线（导入网表、手动走线、打过孔、覆铜）
✅ 设置约束规则（线宽、间距）
✅ 输出 Gerber 和钻孔文件
✅ 读懂并排查 80% 的常见错误

还不够熟练的地方：
- 高频/高速设计约束（差分对、等长）
- 多页原理图（Off-page connector）
- BGA 扇出
- SKILL 脚本自动化

这些可以在实际工作中遇到时针对性地学。
7 天后怎么办？

工具的学习——尤其是 EDA 工具——不是一次性的。7 天课程让你从零到能干活，但真正的熟练是在实际项目中积累的。

建议：
1. 自己定一个小项目（比如你工作中的一块小板子），用 Cadence 完整走一遍
2. 遇到问题先看 Help 文档（F1 键）——Cadence 的帮助文档做得很好
3. 把快捷键表打印出来贴在显示器边上，每天提醒自己用快捷键而不是点菜单
4. 遇到自己解决不了的问题，去 Cadence 官方论坛 community.cadence.com 搜索

有任何问题随时回来讨论。Good luck!
