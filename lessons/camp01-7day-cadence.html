Cadence OrCAD + Allegro 7~14 天学习营

按你自己的节奏走 · 至少 7 天 · 最多 14 天 · 未完成的任务自动顺延

规则说明

每天的任务分为 核心任务（必须完成）和 进阶任务（尽力而为）。
如果当天的核心任务没有完成，未完成的检查项自动顺延到下一天，直到全部打勾。
最快 7 天毕业，最慢 14 天。每天投入约 1~2 小时。

每节课后面的 📘 参考 指向本工坊已有的课程文件，可直接打开对照学习。

📅 Day 1 · 环境准备 + 界面熟悉

🔹 核心任务（必须完成）

任务 1.1：找到并启动 OrCAD Capture

☐ 开始菜单 → 搜索 OrCAD Capture CIS → 打开
☐ 弹出 Product Choices → 选 OrCAD Capture CIS → 勾选 Use as default → OK
☐ 进入主界面后，点 File → Change Product，看看切换窗口长什么样（不切换，Cancel 回到当前）
☐ 菜单栏浏览一遍：File、View、Place、Tools、Reports、Help
📘 参考：app01-cadence-product-choices.html

任务 1.2：找到并启动 Allegro PCB Editor

☐ 开始菜单 → 搜索 Allegro PCB Designer → 打开
☐ Product Choices → 选 Allegro PCB Designer → 把 Options 列表全勾上 → Use as default → OK
☐ 进入主界面后浏览菜单：Setup、Display、Manufacture、Tools

任务 1.3：快捷键初识

☐ 在 OrCAD 中按 Ctrl+N 新建一个空原理图页
☐ 依次按以下键，感受每个键的功能：
P（放器件）→ W（走线）→ F（电源）→ G（地）→ N（网名）→ I（放大）→ O（缩小）→ R（旋转）
☐ 每个键按完按 Esc 退出当前模式
📘 参考：km01-shortcut-memory.html

↓ 如果以上有任一项没完成，明天先补完再学 Day 2 的内容 ↓

🔹 进阶任务（尽力学）

📖 阅读快捷键记忆课程的"Day 1-2 练习法"，在空页面上重复 3 遍 P/W/F/G 序列

📋 Day 1 检查清单

☐能正确选择 Product Choices 并设置 Use as default
☐知道 File → Change Product 可以切换产品特性
☐成功打开过 Allegro PCB Designer
☐在空白原理图页上按出 P/W/F/G/N/I/O/R（不看键盘）

✅ 全部打勾 → 进入 Day 2  |  ❌ 没打完 → 明天继续

📅 Day 2 · OrCAD 原理图基础（上）

🔹 核心任务

任务 2.1：创建项目和放置第一个元件

☐ File → New → Project → 名：DAY2_PRACTICE → 选 Analog or Mixed A/D
☐ 按 P → Add Library → 加载 opamp.olb → 搜索 LM358 → 双击放置
☐ 按 P → 在 DISCRETE.olb 中放置 R（电阻）x 4 个
☐ 放置 C（电容）x 2 个
☐ 按 R（旋转）调整器件方向
📘 参考：pr01-cadence-hands-on.html Step 1~2

任务 2.2：走线连接

☐ 按 W（走线）把 LM358 的 3、4、8 脚连出
☐ 把 4 个电阻和 2 个电容连成如图电路（同相放大器）
☐ 连接完成后检查所有焊盘是否出现红色小圆点（表示已连接）
☐ 按 F 放电源符号（VCC）到 LM358 的 Pin 8
☐ 按 G 放地符号（0）到 LM358 的 Pin 4 和电阻接地端
☐ 按 N 标网络名：输入叫 SIG_IN，输出叫 SIG_OUT
📘 参考：pr01-cadence-hands-on.html Step 4~6

任务 2.3：修改参数并保存

☐ 双击电阻 R1 → 把 Value 从 1k 改为 10k
☐ 其他电阻和电容也改成合理值
☐ 按 Ctrl+S 保存
📘 参考：pr01-cadence-hands-on.html Step 3

🔹 进阶任务

☐ 在信号输出端加一个测试点（Place → Test Point，或放一个排针 Header）
📖 打开 OrCAD 帮助（Help → OrCAD Capture Help）→ 搜索 "Placing parts" 和 "Wiring"

📋 Day 2 检查清单

☐成功创建 Project，选对了 Analog or Mixed A/D
☐会按 P + Add Library 加载库文件
☐会按 W 走线，知道红色圆点表示连接
☐会按 F/G 加电源和地
☐会按 N 修改网络名
☐会按 R 旋转器件
☐会双击修改器件 Value

记得到 Day 2 检查清单 全部打勾了才能进入 Day 3

📅 Day 3 · OrCAD 原理图进阶 + 网表

🔹 核心任务

任务 3.1：创建自己的器件符号

☐ File → New → Library → 保存为 MY_PARTS.olb
☐ 右键 .olb → New Part → Name: OPA381 → Part Reference: U → PCB Footprint: SOIC-8
☐ 用 Place Pin 放 8 个引脚（Pin 1~8），给每个引脚起名（如: OUT, -IN, +IN, VEE, NC, OUT, VCC, NC）
☐ 保存符号
📘 参考：eda01-cadence-orcad-allegro.html 第 1.4 节

任务 3.2：用自建器件替换原理图

☐ 打开 Day 2 的原理图
☐ 按 P → Add Library → 加载 MY_PARTS.olb → 放置 OPA381 替换 LM358
☐ 为每个器件设置 PCB Footprint：R→R0603，C→C0402，OPA381→SOIC-8
📘 参考：pr01-cadence-hands-on.html Step 3

任务 3.3：Annotate + DRC

☐ Tools → Annotate → Incremental → OK（整理位号为 R1,R2,U1 等）
☐ Tools → Design Rules Check → 勾选 Check single node nets → OK
☐ 看 Session Log 底部：如果有错误（红色 X），双击跳到错误位置修复→重复直到 0 Errors
☐ 常见错误：Floating input pin（输入没连）、Single node net（悬空网络）
📘 参考：pr01-cadence-hands-on.html Step 7~8

任务 3.4：生成网表

☐ Tools → Create Netlist → PCB Editor 标签页 → 勾选 Create PCB Editor Netlist → 确定
☐ 看 Session Log：确认显示 Netlist successfully completed
☐ 打开项目文件夹，确认 netlistallegro 文件夹已生成（里面有 .dat, .net, .pst 文件）
📘 参考：pr01-cadence-hands-on.html Step 9

🔹 进阶任务

☐ 在 OPA381 符号的 VCC/VEE 引脚上设置 Power 类型，添加隐藏引脚
☐ 创建一个新器件的符号（如 ADS1256 的简化版），练习批量放引脚

📋 Day 3 检查清单

☐会创建 .olb 库并添加新器件
☐会放置引脚并命名
☐会为器件设置 PCB Footprint 属性
☐跑过 Annotate 整理位号
☐跑过 DRC，零错误通过
☐网表生成成功，netlistallegro 文件夹存在
📅 Day 4 · Allegro 入门 + 导入网表

🔹 核心任务

任务 4.1：创建 Allegro 项目 + 叠层配置

☐ 开始菜单 → Allegro PCB Designer → Product Choices 选 Allegro PCB Designer
☐ File → New → Board → 名：DAY4_BOARD.brd
☐ Setup → Cross-section → 配置为 4 层板：TOP（Signal）/ GND（Plane）/ PWR（Plane）/ BOTTOM（Signal）
☐ 给 GND 层分配网络 GND
📘 参考：pr01-cadence-hands-on.html Step 11

任务 4.2：画板框 + 导入网表

☐ Add → Line → Options: Class=Board Geometry, Subclass=Outline
☐ 画一个 50 x 40 mm 的矩形板框 → 按 F2（Done）
☐ File → Import → Logic → Cadence 标签 → Import directory 选 Day 3 生成的 netlistallegro 文件夹 → Import
☐ 如果报错"Footprint not found"，记下缺失的封装名，去 Setup → User Preferences → Design_paths → padpath/psmpath 设置封装库路径
☐ 导入成功后按 F7 全屏，看到器件散落在板框附近
📘 参考：pr01-cadence-hands-on.html Step 12~13

任务 4.3：摆放器件

☐ Place → Manually → 在 Quick 标签中点 Place All（把所有器件自动放到板框内）
☐ 按 M 移动器件，按 R 旋转
☐ 摆放原则：连接器靠边、IC 居中、电阻电容靠近对应 IC 引脚
📘 参考：pr01-cadence-hands-on.html Step 14

🔹 进阶任务

☐ 设置约束规则：Setup → Constraints → Electrical Constraints → 创建一个 ANALOG 物理约束集，设线宽 0.3mm
☐ 把运放输入输出网络关联到这个约束集

📋 Day 4 检查清单

☐会配置 Cross-section 四层板叠层
☐会画 Board Outline
☐导入网表成功，无 Footprint 报错
☐知道 padpath/psmpath 在哪设置
☐会用 M 移动、R 旋转器件
☐布局整齐，知道摆放顺序
📅 Day 5 · Allegro 布线实战

🔹 核心任务

任务 5.1：手动走关键信号

☐ 按 I（Add Connect），点击 OPA381 输入脚的焊盘开始走线
☐ 走完一段按左键确认，走到终点焊盘再次左键
☐ 走歪了按 Backspace 回退上一步
☐ 需要换层时按 V（打孔），再继续走
☐ 先走模拟信号（运放输入输出）→ 再走电源 → 最后走地线
☐ 走完后按 F2（Done）
📘 参考：pr01-cadence-hands-on.html Step 16

任务 5.2：覆铜

☐ Shape → Add Rect → Options: Class=Etch, Subclass=TOP, Assign net=GND
☐ 在板框内画一个矩形覆盖整个板面（GND 铜皮）
☐ 在运放输入焊盘周围用 Shape → Void → Circular 挖掉铜皮（减少寄生电容）
📘 参考：pr01-cadence-hands-on.html Step 17

🔹 进阶任务

☐ 用 Shift+F5 高亮模拟信号网络，检查走线路径是否合理
☐ 在敏感信号两侧包地（走一条 GND 线在信号线旁边）
☐ 包地线上每隔 5 mm 打一个地过孔（选 GND 网络后按 V）

📋 Day 5 检查清单

☐会用 I 键走线，不会点菜单"Place → Line"那种慢方法
☐会用 Backspace 回退、V 打孔
☐所有网络已走完，PCB 上没有飞线
☐会铺地铜（Shape → Add Rect）
☐会在关键节点下方挖空铜皮
📅 Day 6 · DRC 检查 + 出 Gerber

🔹 核心任务

任务 6.1：DRC 检查

☐ Display → Status → 查看 DRC Errors 是否为 0
☐ 如果有错误，按 Shift+F5 → Options 面板选 DRC Errors → 高亮所有错误
☐ 按 F5 放大，检查每个错误并修改
☐ 重复直到 DRC Errors = 0
📘 参考：pr01-cadence-hands-on.html Step 18

任务 6.2：调整丝印

☐ Manufacture → Silkscreen → 按默认设置生成丝印
☐ 按 M 移动丝印文字（Ref Des）到不覆盖焊盘的位置

任务 6.3：出 Gerber 钻孔文件

☐ Manufacture → Artwork → Film Control → 配置以下 Film：TOP、BOTTOM、SOLDERMASK_TOP、SOLDERMASK_BOTTOM、SILKSCREEN_TOP
☐ 每个 Film 需要先点击那个 Film 名，再在右侧勾选对应的 subclass
☐ 勾选所有 Film → Generate Artwork
☐ Manufacture → NC → Drill → 点击 Drill
☐ 在项目文件夹中找到 artwork/ 目录，确认 .art 和 .drl 文件已生成
📘 参考：pr01-cadence-hands-on.html Step 19

🔹 进阶任务

☐ 用 Allegro 自带的 Artwork Review 打开一个 .art 文件检查
☐ 或用 CAM350 打开检查（如果装了的话）

📋 Day 6 检查清单

☐DRC 零错误
☐丝印没有压在焊盘上
☐Gerber 文件生成成功（.art）
☐钻孔文件生成成功（.drl）
📅 Day 7 · 综合盲走 + Cross-probe + 结业

🔹 核心任务

任务 7.1：Cross-probe 交互定位

☐ 同时打开 OrCAD（Day 3 的原理图）和 Allegro（Day 6 的 PCB）
☐ 在 OrCAD 中选中一个器件 → 按 Alt+Ctrl+F11（或点 Cross-probe 图标）
☐ 看 Allegro 是否自动高亮对应的器件
☐ 反向操作：在 Allegro 中高亮一个网络 → Tools → Cross-probe → 看 OrCAD 中对应的网络高亮
📘 参考：eda01-cadence-orcad-allegro.html 第 3 章

任务 7.2：盲走挑战（全程 45 分钟）

☐ 新建一个 Project，不翻教程，独立完成以下电路：
电路：TIA 光检测前放
BOM：PIN 管 x1, OPA381 x1, 100k 电阻 x1, 0.68pF 电容 x1,
     0.1uF 电容 x2, 10uF 电容 x1, 连接器 x1
流程：创建 Project → 放器件 → 走线 → 电源地 → 网标 → DRC → Netlist
     → Allegro 建板 → 设叠层 → 板框 → 导入网表 → 布局 → 走线 → 覆铜 → DRC → Gerber
☐ 全程不查教程、不查菜单——只用快捷键和肌肉记忆
☐ 完成后再对照 pr01-cadence-hands-on.html 检查每一步是否有遗漏

任务 7.3：自学能力建立

📖 按下 F1 查看 Allegro Help，搜索关键字 Routing，读前 3 条
📖 打开 Cadence 官方社区 community.cadence.com，浏览 PCB Design 板块
📖 在浏览器书签中添加以下链接：Cadence 社区、EDA365 论坛、Cadence 官方帮助文档

📋 Day 7 检查清单

☐会 Cross-probe 在 OrCAD 和 Allegro 之间交互定位
☐45 分钟内盲走完整流程（可以不完美，但不能卡死）
☐知道 F1 搜帮助、知道 community.cadence.com 论坛
☐快捷键已形成肌肉记忆——P/W/F/G/N/I/O/R/M/R/I/V

毕业标准

🎉 全部检查清单打勾 = 毕业！

毕业时你能做到：
1. 独立用 OrCAD 画原理图并生成网表
2. 独立用 Allegro 从零开始布局布线出 Gerber
3. 会 Cross-probe 交互定位
4. 遇到问题知道按 F1 搜帮助、或去社区提问

前三天是最难熬的——快捷键记不住、菜单找不到、封装匹配报错。
但这些都是正常的。每卡住一次，你就多记住一个知识点。
坚持到 Day 7，你会发现前面的付出全都值了。
