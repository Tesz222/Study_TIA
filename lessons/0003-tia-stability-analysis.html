TIA 稳定性分析

Lesson 0003 · 深入 TIA 设计 · 预计用时 40 分钟

本节目标 理解 TIA 自激振荡的根本原因，掌握相位裕度分析方法，学会精确计算反馈电容 Cf，设计出稳定的 TIA 电路。

0. 先解决一个困惑：反馈电容 Cf 究竟是干什么的？

回到 Lesson 0001 的图 2，TIA 电路中 Rf 旁边并联了一个 Cf。我们之前只说它是"用于相位补偿，防止振荡"。但：

  • 为什么没有 Cf 就会振荡？
  • Cf 的值怎么确定？算错了会怎样？
  • 振荡和噪声是一回事吗？

如果你对这三个问题中的任何一个不能立刻给出答案，这节课就是为你准备的。

一个重要的区分

TIA 有两个容易混淆的现象：
振荡（Oscillation）：电路自激，输出出现稳定的周期性波形，通常频率较高——这是稳定性问题，必须避免。
噪声（Noise）：随机的、宽频谱的起伏——可以降低但不能完全消除。

这节课解决的是振荡问题。噪声已经在 Lesson 0002 分析过了。

1. 自激振荡的物理直觉

1.1 一个思想实验

想象你在一个礼堂里，拿着话筒对着音箱讲话：

  • 话筒拾取你的声音 → 音箱放大 → 声音再被话筒拾取 → 再放大……
  • 如果满足两个条件就会发生啸叫（feedback howling）：
    1. 环路增益 ≥ 1：音箱出来的声音足够大，能被话筒有效拾取
    2. 相位相同：音箱出来的声音到达话筒时，和原始声音相位一致（同相叠加）

运放电路的振荡原理完全一样——负反馈变成了正反馈。

巴克豪森判据（Barkhausen Stability Criterion）

一个负反馈系统发生自激振荡的条件是：
在某个频率 fosc 上，环路增益的相移达到 -180°（负反馈变正反馈），且环路增益 ≥ 1（0 dB）。

换句话说：信号在环路里走一圈，不仅没有被衰减，反而被放大，同时相位恰好同相叠加——就振荡了。

1.2 TIA 为什么容易振荡？

回到 TIA 电路。在你的模电知识中，运放电路只要接成负反馈就是稳定的，对吗？大多数情况是的。但 TIA 有一个特殊性：

光电二极管的结电容 CJ 在运放的反相输入端引入了一个额外的极点，增加了相移。

复习一下你的模电知识：

  • 极点：每经过一个极点，增益以 -20 dB/decade 的速率下降，同时引入 -90° 的相移
  • 一个理想运放自身已经有一个主极点（dominant pole）
  • TIA 电路中 CJ（结电容）与运放的开环输出电阻构成了第二个极点
  • 两个极点加起来，总相移在某个频率可以达到 -180°
  • 如果此时环路增益还 > 0 dB，就振荡了

2. 必要的复习：运放的开环增益与频率响应

2.1 运放不是"理想"的

你的模电知识应该还记得：实际运放的开环增益 AOL 不是无穷大，而且随着频率升高而下降。典型运放的频率响应是单极点模型：

AOL(f) = A0 / (1 + j × f / fp)

其中：
A0 = 直流开环增益（典型 100~140 dB）
fp = 运放的主极点频率（通常在几 Hz 到几十 Hz）
GBW = A0 × fp（增益带宽积，恒定值）

这意味着：

  • 在直流附近，增益极高（> 100 dB）
  • 过了主极点 fp 之后，增益以 -20 dB/decade 下降
  • 在频率等于 GBW 时，增益降到 0 dB（单位增益）

2.2 相移回顾

单极点运放产生的最大相移是 -90°。一个单极点运放本身是绝对稳定的——因为它的相移永远达不到 -180°。

问题是：TIA 的反相输入端有 Cin（PD 结电容 CJ + 运放共模输入电容 Ccm），它与运放的开环输出电阻构成了第二个极点，增加额外的 -90° 相移。两个极点 = -180°，于是振荡。

3. TIA 的开环增益分析（关键）

3.1 TIA 的噪声增益（Noise Gain）

在稳定性分析中，我们关心的不是信号增益，而是噪声增益（Noise Gain, NG）——即从运放输入端到输出的增益，它决定环路特性。

TIA 电路的噪声增益由反馈阻抗 Zf 和输入阻抗 Zin 决定：

NG(s) = 1 + Zf(s) / Zin(s)

其中：
Zf(s) = Rf || 1/(s × Cf) → 反馈阻抗
Zin(s) = 1/(s × Cin) → 输入阻抗（忽略 Rsh）

展开后：

NG(f) = (1 + Cin/Cf) × (1 + j × f / fz) / (1 + j × f / fp)

其中：
fz = 1 / (2 × π × Rf × Cf) — 零点
fp = 1 / (2 × π × Rf × (Cin || Cf)) — 极点

3.2 具体例子

考虑一个典型的 TIA 设计：

  • Rf = 100 kΩ
  • Cin = CJ + Ccm = 5 pF + 2 pF = 7 pF
  • 暂定 Cf = 0.5 pF

4. 闭环增益曲线图分析（最重要的图）

这是 TIA 稳定性分析最核心的工具。我们画两条线：

  • 运放的开环增益 AOL：已知的运放特性，从 datasheet 中获得
  • 噪声增益 NG：由电路结构决定（Rf、Cf、Cin）

两条线相交的频率就是环路增益为 1（0 dB）的频率。在这个交点处，如果 AOL 和 NG 之间的闭合速率（closure rate） 是 40 dB/decade，系统就不稳定。

读懂这张图

情况 1：没有 Cf（红色虚线）
- 在 f = 1/(2πRfCin) 处，NG 开始以 +20 dB/dec 上升
- 在交点处，AOL 以 -20 dB/dec 下降，NG 以 +20 dB/dec 上升
- 闭合速率 = 40 dB/dec → 不稳定！

情况 2：有 Cf（绿色实线）
- Cf 引入的零点在 fz 处使 NG 曲线从上升转为平坦
- 在交点处，AOL 以 -20 dB/dec 下降，NG 平坦
- 闭合速率 = 20 dB/dec → 稳定！

5. Cf 的精确计算方法

5.1 目标：足够的相位裕度

什么是相位裕度（Phase Margin, PM）？

  • 在环路增益 = 1（0 dB）的频率（称为 fc，穿越频率）处，总相移与 -180° 之间的差值
  • PM > 45°：系统稳定，阶跃响应有轻微过冲
  • PM > 60°：系统非常稳定，阶跃响应几乎没有过冲
  • PM < 30°：系统接近振荡边缘，阶跃响应有明显振铃
  • PM < 0°：系统振荡

5.2 三步法计算 Cf（工程标准方法）

我们通过一个完整的例子来演示：

设计条件：

运放 OPA381：GBW = 18 MHz，A0 = 130 dB

光电二极管：CJ = 3 pF，运放 Ccm = 2 pF，Cin = 5 pF

Rf = 100 kΩ，目标相位裕度 = 60°

  1. 第一步：计算不加 Cf 时的穿越频率和相位裕度
    无 Cf 时，噪声增益的零点在：
    fz = 1 / (2πRfCin) = 1 / (6.28 × 1E5 × 5E-12) = 318 kHz

    穿越频率 fc = √(GBW / (2πRfCin))
    = √(18E6 / (6.28 × 1E5 × 5E-12)) = √(5.73E12) ≈ 2.4 MHz

    此时：
    - 运放自身贡献的相移 ≈ -90°
    - Cin 贡献的相移 ≈ -90°
    - 总相移 ≈ -180°，相位裕度 ≈ 0° → 振荡！
  2. 第二步：计算所需的 Cf
    标准公式：
    Cf = √(Cin / (2πRf × GBW))
    = √(5E-12 / (6.28 × 1E5 × 18E6))
    = √(4.42E-25) = 0.66 pF

    选择标准值：Cf = 0.68 pF（E12 系列）
  3. 第三步：验证最终的相位裕度
    加上 Cf = 0.68 pF 后：

    fz = 1 / (2πRfCf)
    = 1 / (6.28 × 1E5 × 0.68E-12) = 2.34 MHz

    高频噪声增益：
    NGhigh = 1 + Cin/Cf = 1 + 5/0.68 = 8.35
    用 dB 表示：20 × log10(8.35) = 18.4 dB

    新的穿越频率：
    fc = GBW / 10^(18.4/20) = 18M / 8.35 = 2.16 MHz

    预估 PM ≈ 55° ~ 65° 稳定！

6. 工程速算法

工程速算法

Cf = √(Cin / (2πRf × GBW))

在大多数常见情况下（Rf 在 10 k ~ 1 MΩ 范围，Cin 在 1 ~ 10 pF 范围），也可以直接用：

Cf ≈ 0.1 ~ 1 pF，然后通过实际测试调整。

测试方法：输入一个阶跃光信号，观察输出波形。如果：
- 有过冲和振铃 → Cf 偏小，需要增大
- 响应太慢（上升沿太缓）→ Cf 偏大，需要减小
- 干净利落的阶跃响应 → Cf 合适

7. 完整的 TIA 稳定性设计流程

TIA 稳定性设计六步法

  1. 确定已知参数：Rf（由增益需求决定），Cin = CJ + Ccm + Cdiff（PD 结电容 + 运放输入电容），运放 GBW
  2. 检查是否需要 Cf：如果不加 Cf 时的穿越频率 fc 处的相位裕度 > 45°，则不需要。但通常 TIA 都需要。
  3. 计算 Cf 初始值：使用公式 Cf = √(Cin / (2πRf × GBW))
  4. 选择标准值：根据 E12/E24 系列选择最接近的标准电容值
  5. 仿真验证：使用 TINA-TI 或 LTspice 仿真环路增益的相位裕度
  6. 实物测试：输入阶跃信号，观察输出波形，微调 Cf
常见陷阱

陷阱 1：Cf 算出来是 0.3 pF，但 PCB 上 0.3 pF 的寄生电容已经比这大了！
→ 如果计算值小于 0.5 pF，说明 PCB 布局的寄生电容可能已经提供了足够的补偿。先不加 Cf，测试稳定性后再决定。

陷阱 2：Cin 中忽略了运放的差分输入电容 Cdiff
→ 实际 Cin = CJ + Ccm + Cdiff，不要漏算 Cdiff（通常在 1~3 pF）

陷阱 3：Rf 两端的 PCB 焊盘和走线也会引入寄生电容
→ 0603 封装的电阻本身的寄生电容约 0.05 pF，0402 更小

8. 动手练习

练习 1：计算 Cf

你正在设计一个光功率计的 TIA 电路：

  • Rf = 200 kΩ
  • PD：CJ = 2 pF，R = 0.9 A/W
  • 运放：OPA657（GBW = 1.6 GHz，Ccm = 4 pF，Cdiff = 2 pF）

计算所需的最小 Cf 值。

解题过程：

Step 1：计算总输入电容
Cin = CJ + Ccm + Cdiff = 2 + 4 + 2 = 8 pF

Step 2：应用公式
Cf = √(Cin / (2πRf × GBW))
= √(8E-12 / (6.28 × 2E5 × 1.6E9))
= √(3.98E-27) = 0.063 pF（63 fF）

结论：这个值非常小，PCB 本身的寄生电容可能已经足够。可以先用阶跃响应测试确认稳定性。

练习 2：判断稳定性

一个 TIA 的阶跃响应如下图所示。判断 Cf 是否合适，应该增大还是减小？

场景 A：阶跃后输出快速上升，但有过冲和几次衰减振荡（振铃）
场景 B：阶跃后输出缓慢爬升，达到稳定的时间很长
场景 C：输出呈现稳定的正弦波，即使没有输入信号
场景 D：阶跃后输出干净利落地上升到目标值，几乎没有过冲

分析：

场景 A：Cf 偏小。有振铃说明相位裕度不足（可能在 30~45° 范围）。增大 Cf。

场景 B：Cf 偏大。过补偿导致带宽太窄，响应变慢。减小 Cf。

场景 C：自激振荡！相位裕度接近 0°。需要显著增大 Cf。

场景 D：Cf 合适。相位裕度约 60°，响应速度和稳定性平衡良好。

课程总结

TIA 稳定性分析的核心逻辑链：

CJ（结电容）→ 引入额外极点 → 总相移 -180° → 振荡
Cf（反馈电容）→ 引入零点补偿 → 降低闭合速率 → 稳定

三个关键设计参数：Cin、Rf、GBW，共同决定 Cf 的值。
