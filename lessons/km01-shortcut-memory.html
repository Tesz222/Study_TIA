快捷键记忆：OrCAD + Allegro 核心快捷键

Lesson KM01 · 工具效率 · 预计用时 15 分钟 + 日常练习

记忆目标 掌握 OrCAD Capture 和 Allegro PCB Editor 中最高频使用的快捷键，按功能分组记忆，配合练习肌肉记忆。目标是：不看菜单，不碰鼠标——纯键盘操作 80% 的常用操作。

OrCAD 快捷键（共 8 个核心键）

放置类 — 按字母联想（P / W / N / F / G）
PPlace Part放置器件P = Part
WPlace Wire走线W = Wire
NPlace Net Alias标注网络名N = Net
FPlace Power放置电源符号F = Force/Power
GPlace Ground放置地符号G = Ground
编辑与导航类
RRotate旋转器件（放置时按）R = Rotate
IZoom In放大I = In
OZoom Out缩小O = Out
EZoom Edit / Ctrl+E 打开属性编辑器件属性E = Edit
P W N F G

记忆口诀："P 放器件 W 走线，N 标网名 F/G 电源地"

I O R

记忆口诀："I 放大 O 缩小，R 旋转少不了"

Allegro 快捷键（核心 10 键 + 功能键 6 个）

最最核心 — 布局布线的灵魂（M / I / R / V / Delete）

IAdd Connect走线（核心操作！）I 像"I connect"
RRotate旋转R = Rotate
VAdd Via走线中打孔V = Via

M I R V Delete

记忆口诀："M 移动 I 走线，R 旋转 V 打孔，删掉 Delete 不手软"

功能键 — 视图与测量（F5 / F6 / F7 / F3 / F4）

F6Zoom Out缩小

F5 F6 F7 F3 F4

高亮与刷新

Shift+F6Dehighlight取消高亮

Shift+F5 Shift+F6

进阶操作 — 走线中的辅助键

BackspaceUndo Last走线时取消上一步走线走错了，不用退出，按 Backspace 回退一段

Shift+拖动Lock Axis锁定水平/垂直方向走线时按住 Shift 再拖
Ctrl+ASelect All全选
Ctrl+SSave保存经常按！

记忆卡片：每日练习法

Day 1-2：OrCAD 基本操作

每天花 3 分钟，打开 OrCAD 新建一个空白原理图页，重复以下动作 5 遍：

  1. 按 P 打开 Place Part → 选一个电阻 → 放置 → 按 Esc 退出
  2. 按 W 走线 → 在两个焊盘之间走一条线 → 按 Esc
  3. 按 F 放电源 → 按 G 放地
  4. 按 I/O 放大缩小 → 按 R 旋转

目标：这 5 个键不用想就能按出来

Day 3-4：Allegro 核心 5 键

打开之前做的 TIA_BOARD.brd 文件：

  1. 按 M 移动一个电阻 → 按 R 旋转 → 放回原位 → F2 结束
  2. 按 I 开始走线 → 走一小段 → 按 V 打个孔 → 继续走 → F2
  3. 按 Backspace 回退一步（练习回退的感觉）
  4. 按 F5/F6/F7 练习视图控制

目标：M / I / R / V / Delete 不用想就能按

Day 5-6：组合操作

  1. 打开原理图：P 放器件 → W 走线 → N 标网名 → Ctrl+S 保存
  2. 打开 PCB：I 走一段信号线 → 走歪了按 Backspace 回退
  3. 按 Shift+F5 高亮 TIA_OUT 网络 → 按 Shift+F6 取消
  4. 按 ` 切换透明度看底层

目标：能连续按键完成一个完整的操作序列

Day 7：全流程计时挑战

从零开始，计时完成以下操作序列：

  1. OrCAD 打开 → 按 P 放一个运放 + 几个电阻电容
  2. 按 W 走通
  3. 按 F/G 加电源地
  4. 按 Ctrl+S 保存
  5. 切换到 Allegro → F7 全屏显示
  6. 按 M 调整个器件位置 → R 旋转
  7. 按 I 走一段线
  8. 按 Ctrl+S 保存

目标：能在 2 分钟内完成以上所有操作，全程不碰鼠标

记忆策略总结

优先记忆（每天必用）：
OrCAD：P W F G R — 5 个键完成 80% 的原理图操作
Allegro：M I R V F5 F6 F7 — 7 个键完成 80% 的 PCB 操作

记忆方法：不要死记硬背，而是用的时候想三秒——第一次按键盘快捷键可能比点菜单慢，但第二次就一样快，第三次就更快了。一周后就不需要想了。
