可变光衰减器 (VOA) 基础

光通信测试仪表系列 · Lesson 0007

关键洞察

VOA 是光测试仪表中唯一一个同时出现在发射链路和接收链路的器件——它既用于校准接收机，也用于模拟光纤损耗。掌握了 VOA，你就掌握了光测试系统中"可控衰减"这个核心自由度。

1. 为什么需要 VOA

在前面的课程中，你完整学习了光电接收前端：光信号 → 光电二极管 → TIA → ADC → 数字码。但测试仪表还需要"可控光源"这一端：

  • 光功率计校准：需要已知且精确可调的光功率作为参考
  • 接收机灵敏度测试：不断降低光功率直到误码率达到阈值
  • 链路损耗模拟：在实验室中复现实际光纤链路的衰减特性
  • 通道均衡：WDM 系统中各通道功率对齐

所有这些场景都需要一个能精确、稳定、可重复地控制光功率的器件——这就是 VOA（Variable Optical Attenuator）。

2. 工作原理与技术路线

VOA 的核心物理思想很简单：在光路中引入可控的插入损耗。但实现方式各有不同，每种技术都有自己的工程权衡。

2.1 MEMS VOA（主流）

MEMS（微机电系统）VOA 是当前最主流的技术路线。其工作原理是：

  • 通过静电驱动或电磁驱动使微镜偏转
  • 偏转角度改变光纤耦合效率，从而控制衰减量
  • 典型偏转角度范围 0~10°，对应衰减范围 0~60 dB

图1: MEMS VOA 基本原理 — 微镜偏转改变耦合效率

2.2 磁光 VOA

基于法拉第效应：外加磁场改变磁光晶体的偏振旋转角度，通过偏振器转换成可调的衰减量。

  • 优点：全固态，无运动部件，响应速度快（亚毫秒级）
  • 典型应用：通道功率均衡（WDM 系统中需要快速响应）
  • 局限性：PDL 相对较大，成本较高

2.3 热光 VOA（波导型）

基于平面光波导（PLC）技术，通过加热改变波导材料的折射率，利用马赫-曾德尔干涉仪或定向耦合器实现衰减。

  • 优点：易于集成，适合多通道 VOA 阵列
  • 缺点：响应速度慢（毫秒级），温控依赖度高
  • 典型场景：WDM 通道均衡

2.4 技术路线对比

MEMS VOA

衰减范围: 0~60 dB

响应时间: ~5 ms

+ 宽衰减范围

+ 低插入损耗 (<1 dB)

+ 低 PDL

— 有运动部件，可靠性受限

— 对振动敏感

磁光 VOA

衰减范围: 0~40 dB

响应时间: <0.5 ms

+ 固态，可靠性高

+ 响应极快

— PDL 较大

— 成本较高

热光 VOA (PLC)

衰减范围: 0~30 dB

响应时间: ~10 ms

+ 可多通道集成

+ 无运动部件

— 功耗较高

— 温度敏感

液晶 VOA

衰减范围: 0~50 dB

+ 驱动简单

— 温度稳定性受限

— 波长相关损耗较大

工程连线：在光通信测试仪表中，MEMS VOA 占据主导地位。你可能会问——为什么是 MEMS？因为测试仪表最看重 衰减范围 + 精度 + 重复性，MEMS 恰好在这三个维度上综合最优。

3. 关键性能指标

VOA 的数据手册中有几个核心参数，读懂了它们就等于读懂了 VOA：

参数 含义 典型值（测试级）
衰减范围 最小到最大衰减量的调节范围 0~60 dB
分辨率 可调节的最小步进 0.01 dB（测试级）
插入损耗 (IL) 衰减量为 0 时的固有损耗 <1.0 dB
回波损耗 (RL) 反射光功率与入射光功率之比 >45 dB（测试级要求）
PDL 偏振相关损耗，不同偏振态的衰减差异 <0.1 dB（测试级）
WDL 波长相关损耗，不同波长的衰减差异 <0.3 dB（C 波段）
响应时间 从控制电压变化到衰减值稳定所需时间 <10 ms
光学功率处理能力 可承受的最大输入光功率 200~500 mW
重复性 同设置下多次测量的衰减一致性 ±0.02 dB（测试级）
⚠ 测试级 vs 通信级

光通信网络中的 VOA 指标通常宽松得多（如 PDL <0.3 dB、分辨率 0.1 dB），而测试仪表级 VOA 的要求要严格 5~10 倍。这是为什么测试级 VOA 的价格可以是通信级的 10 倍以上。

4. 光功率控制环路

现在让我们把你已学的信号链与 VOA 连接起来。测试仪表中最常见的配置是闭环光功率控制：

┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐ │ 光源 │────▶ │ VOA │────▶ │ 抽头耦合器 │────▶ │ 输出 │ │ (激光器) │ │ │ │ (90/10) │ │ │ └──────────┘ └──────────┘ └──────────┘ └──────────┘ ▲ │ (10%) │ ▼ ┌──────────┐ ┌──────────┐ │ DAC │◀────│ MCU │◀────│ ADC │◀────│ TIA+PD │ │ + 驱动 │ │ (PID) │ │ │ │ (监测) │ └──────────┘ └──────────┘ └──────────┘ └────────┘
图2: 闭环光功率控制系统 — 你已学的 TIA+ADC 在这个环路中作为测量反馈支路

这个环路的设计从你的已有知识出发：

  • 监测支路（你已掌握）：抽头光 → PIN/TIA → ADC → MCU
  • 控制支路（本节新知识）：MCU → DAC → VOA 驱动器 → MEMS 偏转
  • 控制算法：PID 环路，将测量功率与设定点比较，调整 VOA 驱动电压

4.1 驱动电路设计要点

以 MEMS VOA 为例，其驱动电路的工程设计要点：

  • 驱动电压：典型 MEMS 需要 0~120 V（静电驱动），需要 DC-DC 升压 + 运放缓冲
  • 分辨率：16-bit DAC 配合 120 V 驱动范围，理论步进 = 120/65536 ≈ 1.8 mV，对应衰减分辨率约 0.005 dB
  • 纹波与噪声：驱动电压上的任何纹波都会直接调制到输出光功率上。1 mVpp 纹波在典型 MEMS 上可能导致 ~0.003 dB 功率抖动
  • 建立时间：DAC + 高压放大器的建立时间决定环路的带宽上限
🔗 与你已学的连接

在 Lesson 0004（TIA PCB 布局）中你学习了如何在 pA 级电流下控制漏电流。VOA 驱动电路处在另一极——高电压（120 V）下防止击穿和爬电。这是 PCB 布局的另一个极端：高电压间距 > 低噪声设计。典型 120 V 走线间距要求 ≥1.5 mm（2 层 FR4）。

5. 测试仪表中的典型应用

5.1 光功率计校准

使用标准光源 + VOA 产生覆盖全量程的功率点。校准流程：设置 VOA 至特定衰减 → 读取参考光功率计 → 读取被校仪表 → 记录偏差。VOA 在此处的精度决定校准的阶梯精度。

5.2 接收机灵敏度测试

ITU-T 建议的灵敏度测试标准方法：固定发射功率不变，逐步增大 VOA 衰减直到 BER 达到阈值（如 10-12）。VOA 的分辨率和重复性是关键——你需要以 0.1 dB 步进精确扫描灵敏度拐点。

5.3 WDM 通道功率均衡

多通道 VOA 阵列（通常是热光 PLC 型）配合 PD 阵列，由 DSP 控制每个通道的衰减量，使各通道输出功率一致。

6. 选型考虑 — 测试仪表视角

假设你正在设计一台光功率计校准用的可调光衰减模块，选型优先级建议：

  1. 技术路线：MEMS VOA（衰减范围、精度、成本综合最优）
  2. 衰减范围：0~60 dB（覆盖绝大多数测试场景）
  3. 分辨率：≤0.01 dB（对应 16-bit DAC 加适当驱动设计）
  4. PDL：<0.05 dB（测试级功率计必须考虑偏振影响）
  5. 回波损耗：>50 dB（避免反射影响光源稳定性）
  6. 光功率处理能力：≥200 mW（+23 dBm，覆盖通用测试场景）
💡 工程直觉

当你说"我需要一个测试级 VOA"时，行业里默认就是 MEMS VOA。这是经过 20 多年市场验证的选择。除非有极高速响应（磁光）或多通道集成（PLC）的特殊需求，否则 MEMS 总是起点。

7. 练习与自测

问题 1：在闭环光功率控制系统中，VOA 的正常工作条件是以下哪一项？

A. VOA 的衰减范围为 0~60 dB
B. VOA 的响应速度快于环路更新速率
C. VOA 的插入损耗优于 0.5 dB
D. VOA 的 PDL 小于 0.1 dB
答案：B
VOA 的响应速度决定了闭环控制系统的带宽上限。如果 VOA 响应滞后于环路更新，系统会振荡。其他选项都是重要指标，但唯有响应时间是闭环稳定性的必要条件。复习图 2 中的控制环路来理解这个约束。

问题 2：你在设计一个接收机灵敏度测试系统，需要在 -10 dBm 到 -40 dBm 范围内以 0.1 dB 步进精确调节。光源输出为 +3 dBm。VOA 需要多大的衰减范围？

A. 10 dB
B. 13 dB
C. 43 dB
D. 60 dB
答案：C
最低输出 = -40 dBm，最高输出 = -10 dBm，需要覆盖范围 = 30 dB。但输入功率是 +3 dBm，所以最小衰减 = 3 - (-10) = 13 dB，最大衰减 = 3 - (-40) = 43 dB。因此需要至少 13~43 dB（范围 ≥30 dB）。43 dB 正好覆盖。提示：别忘了加上插入损耗的余量（通常 +2~3 dB）。

问题 3：以下哪种 VOA 技术路线最适合需要 <1 ms 响应时间 的通道功率均衡应用？

A. MEMS VOA
B. 磁光 VOA
C. 热光 PLC VOA
D. 液晶 VOA
答案：B
磁光 VOA 的固态机制可实现 <0.5 ms 响应时间。MEMS（~5 ms）、热光（~10 ms）和液晶（~20 ms）都达不到 <1 ms 的要求。但注意磁光 VOA 的 PDL 通常较大，设计中需要考虑偏振管理。

8. 小结

  • VOA 是光测试仪表中"可控衰减"自由度的实现者
  • MEMS VOA 是测试级仪表的默认技术路线，兼顾衰减范围、精度和成本
  • 闭环控制 将 VOA 与你已学的 TIA/ADC 信号链整合为完整的测试系统
  • 测试级 VOA 的指标要求远高于通信级——PDL、回波损耗、重复性是核心关注点
  • VOA 驱动电路设计 的挑战与 TIA 不同：高压管理取代了低漏电流管理
