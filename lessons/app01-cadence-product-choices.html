Cadence 17.4 产品选型指南

Appendix · 解决"该选哪个？"的问题

问题：每次打开 Cadence 17.4，弹出 Product Choices 窗口，几十个选项不知道选哪个。
答案：选错了也能用，但功能可能受限（比如没有 CIS、没有 PSpice、没有 Allegro 高级功能）。本文告诉你每个选项干什么，以及你应该选哪个。

0. 理解 Cadence 的产品体系

Cadence 把同样的软件内核，按功能层级切分成不同的产品套餐。从低到高大致是：

OrCAD Capture        → 基础原理图（入门级）
OrCAD Capture CIS    → 原理图 + 元件库管理（CIS = Component Information System）
Allegro Design Entry → Allegro 版的原理图（功能更多但界面一样）

OrCAD PCB Designer         → 基础 PCB 布局（入门级）
OrCAD PCB Designer Pro     → 中端 PCB 布局
Allegro PCB Designer        → Allegro 版 PCB 布局（功能最全）
Allegro PCB Designer Pro   → 高速/高级功能
Allegro PCB Designer XL    → 最高级，含团队协作

简单来说：OrCAD = 小公司/个人用的精简版；Allegro = 大公司/专业用的完全版。操作界面几乎一样，但 Allegro 版本开放了更多高级功能。

两个维度的理解

维度 1：OrCAD vs Allegro
- OrCAD 系列：低门槛，功能精简，价格低廉
- Allegro 系列：全功能，支持复杂 PCB（高密、高速、射频），价格高

维度 2：有无 CIS
- 无 CIS：手动管理元器件库，适合小项目
- 有 CIS：连接公司数据库管理元器件，适合企业级、多人协作

维度 3：有无 PSpice
- 无 PSpice：只能画原理图，不能仿真
- 有 PSpice：可以在原理图中仿真模拟电路

1. 每次开机弹出的 Product Choices 是什么意思？

当你在开始菜单中点击 OrCAD Capture CIS 或 Allegro PCB Designer 时，实际上你启动的是一个统一的 License Manager，它让你选择用哪个产品套餐（Suite）来启动当前工具。

你可以把它理解为：

  • "我要启动 OrCAD Capture 这个程序"
  • "但是……我想让它以 OrCAD Capture 的模式运行？还是以 Allegro Design Entry 的模式运行？"
  • "是否要同时加载 CIS 功能（元器件数据库管理）？"
  • "是否要同时加载 PSpice 功能（仿真）？"

选不同的选项，同一个程序启用的功能集不同。

2. 选项分类详解

2.1 按是否有 CIS（是否管理元器件数据库）

选项名中带有含义谁需要
CISComponent Information System
元器件信息管理系统		公司有统一元器件库，需要用数据库管理器件（封装、料号、供应商）
无 CIS纯粹画原理图个人使用、小团队、不需要管理数据库
不建议：如果你的公司不需要数据库管理，不要选 "and CIS" 的版本——加载更慢，而且会多出一堆"配置 CIS 数据库"的提示。

2.2 按是否有 PSpice（是否做仿真）

选项名中带有含义谁需要
PSpice模拟电路仿真功能画完原理图后需要仿真验证的人
无 PSpice纯粹画原理图不需要仿真，只负责设计

2.3 按 OrCAD vs Allegro（功能级别）

系列原理图工具名对应的 PCB 工具名区别
OrCADOrCAD CaptureOrCAD PCB Designer基础功能，适合 ≤ 4 层板、中低速
OrCAD ProOrCAD Capture ProOrCAD PCB Designer Professional中等功能，支持更多层、更高速度
AllegroAllegro Design EntryAllegro PCB Designer全功能，高速/射频/高密

3. 逐条分析你的选项列表

针对你列出的选项，逐条分析：

1. OrCAD Capture

含义：最基本的原理图编辑。无 CIS，无 PSpice。

适合：个人画简单原理图。启动最快。

注意：生成的网表只能导入 OrCAD PCB Designer，不能导入 Allegro PCB Designer（除非你另买 License）。

2. OrCAD Unison PCB

含义：OrCAD Capture + OrCAD PCB Designer 的组合套餐。一次启动，同时拥有原理图和 PCB 布局功能。

适合：如果公司买了 Unison 套装，你就选这个，不用分别启动。

3. OrCAD PSpice Designer

含义：原理图 + PSpice 仿真。无 CIS。

适合：需要仿真模拟电路（运放、电源、滤波器）但不需元器件数据库管理。

4. OrCAD Unison Ultra

含义：OrCAD 最高级套餐，含 Capture + PCB + PSpice + 高级分析。

适合：对仿真和 PCB 都有需求的公司。

5. OrCAD PSpice TI

含义：TI 定制的 PSpice 版本，只能仿真 TI 的器件。

适合：只在用 TI 芯片做仿真时选这个。平时画原理图不要选它。

6. OrCAD PCB Designer Professional

含义：OrCAD 中端 PCB 布局工具。

适合：做 PCB 布局时用。比基础版支持更多层、更复杂板卡。

7. OrCAD PCB Designer Professional w/PSpice

含义：同上，加上了 PSpice 仿真。

8. OrCAD PSpice Designer Plus

含义：PSpice 高级版本，含高级分析（蒙特卡洛、灵敏度分析等）。

9. Capture and CIS

含义：Capture 加上元器件数据库管理功能。

适合：公司有统一的元器件库（通常用 Access 数据库或 SQL 数据库管理）。

10~14: ... and CIS

含义：前面各版本加上 CIS 功能。CIS 只是附加功能，其他和对应版本一样。

15. OrCAD Capture CIS

含义：这是最常用的原理图选项：Capture + CIS（元件库管理）的组合。

16. Allegro Design Entry CIS

含义：Allegro 版的 Capture，带有 CIS。注意这是原理图工具，不是 PCB 工具。

适合：公司买的 Allegro 套装，用 Allegro 版的原理图。

17. Allegro PCB Design CIS L

含义：Allegro PCB 布局工具的入门版（L = Lite），带有 CIS。

18. Allegro PCB Librarian XL

含义：Allegro 封装库管理工具。不是画原理图也不是画 PCB 的——是用来创建和管理 PCB 封装的。

19. OrCAD PCB Designer Standard and CIS

含义：OrCAD PCB 标准版，带 CIS。

适合：有元器件库管理需求的 PCB 设计。

4. 你应该选哪个？

场景一：画原理图

选：OrCAD Capture CIS（第 15 项）

理由：Capture 带 CIS 是最通用的原理图选择。如果你的公司不需要元器件数据库，可以退一步选 OrCAD Capture（第 1 项），启动更快。

注意：选 OrCAD Capture 还是 Allegro Design Entry？

如果你的公司买的是 Allegro 套装（你同时也要用 Allegro PCB 做布局），选 Allegro Design Entry CIS（第 16 项）

如果你的公司买的是 OrCAD 套装（你用 OrCAD PCB 做布局），选 OrCAD Capture CIS（第 15 项）

你不知道公司买的是哪个？问同事。如果问不到：先选 OrCAD Capture CIS。如果后面发现功能不够、或者跑不出网表换 Allegro，再选 Allegro Design Entry CIS。

场景二：画原理图 + 做仿真

选：OrCAD PSpice Designer（第 3 项）或对应带 CIS 的版本

如果你需要在画完原理图后用 PSpice 做瞬态仿真、AC 分析、DC 扫描，选中带 PSpice 的选项。

场景三：做 PCB 布局

选：OrCAD PCB Designer Professional（第 6 项）或 Allegro PCB Designer（对应项）

做 PCB 布局时，在开始菜单中打开 OrCAD PCB Designer 或 Allegro PCB Designer（不是 Capture），同样会弹出 Product Choices。从中选择 PCB 相关的版本。

重要：OrCAD 和 Allegro 的 PCB 工具不能混搭

OrCAD Capture 生成的网表 → 只能导入 OrCAD PCB Designer
Allegro Design Entry 生成的网表 → 只能导入 Allegro PCB Designer

不能串：用 OrCAD Capture 画原理图，然后导入 Allegro PCB Designer —— 文件格式不兼容。

但：Allegro Design Entry 生成的网表可以导入 OrCAD PCB Designer（向下兼容）。

所以如果公司同时有 OrCAD 和 Allegro 的 License，统一用 Allegro Design Entry 画原理图是最稳妥的——生成的网表两个 PCB 工具都能吃。

场景四：管理封装库

选：Allegro PCB Librarian XL（第 18 项）

创建和编辑 PCB 封装（.dra 文件）的工具。不是画板子的，是建库的。

6. 关于 "Allegro 怎么选你没写"

Allegro 和 OrCAD 的 Product Choices 逻辑完全一样。你在教材中看到我写"OrCAD Capture 里怎么画原理图"，但这些操作在 Allegro Design Entry 里界面完全一样、快捷键完全一样。

唯一的区别是启动时的选择：

  • 画原理图：开始菜单 → Allegro Design Entry CIS（或 Allegro Design Entry） → Product Choices 选 Allegro Design Entry CIS
  • 画 PCB：开始菜单 → Allegro PCB Designer → Product Choices 选 Allegro PCB Designer 或对应版本
  • 建封装：开始菜单 → Allegro PCB Librarian → Product Choices 选 Allegro PCB Librarian XL

界面和快捷键——我在实操课里写的所有 P W I M R V——在 OrCAD 和 Allegro 中完全一样，因为底层是同一个程序。

7. 快速决策树（1 分钟判断）

你公司买的是 OrCAD 还是 Allegro？
   │
   ├─ 不知道 → 问同事。等回复时先选 OrCAD Capture CIS
   │
   ├─ OrCAD →
   │     ├─ 画原理图 → OrCAD Capture CIS
   │     ├─ 做仿真   → OrCAD PSpice Designer
   │     └─ 画 PCB   → OrCAD PCB Designer Professional
   │
   └─ Allegro →
         ├─ 画原理图 → Allegro Design Entry CIS
         ├─ 画 PCB   → Allegro PCB Designer
         └─ 建封装   → Allegro PCB Librarian XL
一句话总结

还在纠结的时候 → 选 OrCAD Capture CIS 画原理图，选 OrCAD PCB Designer Professional 画 PCB。
这两个选项覆盖了 90% 的模拟硬件工程师日常需求。

等你发现功能不够用（比如要做高速信号、要团队协作），自然会知道该升级到哪个 Allegro 版本。
