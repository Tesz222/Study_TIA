四通道 VOA 控制电路 — 详细连接说明

配套 Lesson 0012 · 文字版连接描述

说明

这份文档用纯文字描述每条信号路径，告诉你"哪颗芯片的哪个引脚，通过什么值的电阻/电容，连到哪颗芯片的哪个引脚"。这是 Lesson 0012 中电路图的逐句翻译。

一、电源连接

1.1 +5VA 模拟电源路径

输入：外部 +5V 电源进入 PCB，首先到达铁氧体磁珠 FB1 的输入端（引脚 1）。

FB1：型号 BLM18PG121SN1，0603 封装，阻抗 120 Ω @ 100 MHz，DCR = 0.15 Ω，额定电流 300 mA。

C1 (10 µF/16V X7R)：一端接 FB1 的输入端（与外部电源同一侧），另一端接 GND。

C2 (10 µF/16V X7R)：与 C1 并联，同样一端接 FB1 输入端，另一端接 GND。C1+C2 构成输入储能。

FB1 输出端（引脚 2）：输出滤波后的 +5VA 电源轨。后面接：

—— C3 (100 nF/16V X5R)：一端接 FB1 输出端，另一端接 GND。提供高频去耦。

—— 然后 +5VA 分别去到以下 IC 的电源引脚：

  • DAC8568 (U1) 引脚 16 (VDD)：直接接 +5VA
  • OPA381 (U2) 引脚 7 (V+)：直接接 +5VA
  • OPA2705 #1 (U3) 引脚 8 (V+)：直接接 +5VA
  • OPA2705 #2 (U4) 引脚 8 (V+)：直接接 +5VA
  • OPA2705 #3 (U5) 引脚 8 (V+)：直接接 +5VA

每个 IC 的 V+ 引脚紧邻处各有一颗 100 nF 去耦电容到 GND（C7、C12、C15、C18、C21、C23、C25、C29）。

1.2 +3.3VD 数字电源路径

FB2：型号 BLM18AG221SN1，0603 封装，阻抗 30 Ω @ 100 MHz，DCR = 0.05 Ω，额定电流 500 mA。

C4 (10 µF/10V X7R)：接 FB2 输入端到 GND。

C5 (0.1 µF/16V X5R)：与 C4 并联，接 FB2 输入端到 GND。

FB2 输出端：输出 +3.3VD 电源轨。

C6 (100 nF/16V X5R)：接 FB2 输出端到 GND。

+3.3VD 只供应 MCU 的数字电源和 SPI 上拉电阻 R1/R4/R5 的上拉端。不进入模拟区域。

1.3 GND 连接

所有 GND 符号最终在 PCB 的模拟地平面（AGND）上星形汇集。

DAC8568 VREFGND (U1-4)：单独走线（宽 0.5 mm），在 U1 下方单点连接到 AGND。这条走线不与其他 GND 共用。

其余所有 GND：各 IC 的 GND 引脚直接打孔到 AGND 平面。

模拟地和数字地不在 PCB 上分割——使用统一地平面，但物理分区（模拟器件在左，数字器件在右）。

二、DAC8568 (U1) 连接

2.1 电源和基准

U1-16 (VDD) → 接 +5VA（FB1 滤波后的模拟电源）。

U1-5 (GND) → 接 AGND。

U1-3 (VREF) → 接 C9 (100 nF/50V NP0/C0G) 的一端。C9 的另一端接 U1-4 (VREFGND)。使用内部基准时，VREF 引脚输出 2.5 V。C9 补偿基准缓冲器，必用 NP0 材质（防止 DC 偏置下容值下降）。

U1-4 (VREFGND) → 单独走线到 AGND 单点连接点。

2.2 SPI 数字接口

U1-11 (SYNC/CS) → 接 R1 (10 kΩ, ±5%) 的一端。R1 的另一端接 +3.3VD（上拉至数字电源）。同时 U1-11 也接到 MCU 的 GPIO 引脚（CS_DAC）。

CS 在空闲时必须为高电平。10 kΩ 上拉确保 MCU 未初始化时 CS 不浮空。

U1-12 (SCLK) → 串接 R2 (22 Ω, ±5%) 后，接到 MCU 的 SPI1_SCLK 引脚。

R2 靠近 U1 放置，抑制 SCLK 信号反射和振铃。

U1-13 (DIN) → 串接 R3 (22 Ω, ±5%) 后，接到 MCU 的 SPI1_MOSI 引脚。

U1-14 (LDAC) → 接 R4 (10 kΩ, ±5%) 的一端。R4 的另一端接 GND（下拉到地）。

LDAC 永久低电平 = DAC 寄存器在每个 SPI 写操作完成后立即更新。如果不需要永久更新，可将 R4 改为接 MCU GPIO。

U1-15 (CLR) → 接 R5 (10 kΩ, ±5%) 的一端。R5 的另一端接 +3.3VD（上拉，禁用异步清零）。

2.3 DAC 输出通道（仅使用 A~D）

U1-1 (VOUT A) → 串接 R6 (100 Ω, ±1%) → 接到 OPA2705 #1 的 U3A-3 (+IN)。同时，U3A-3 到 GND 之间并联 C10 (1 nF/50V NP0)。

R6 + C10 构成低通滤波器，转折频率 f_c = 1/(2π×100Ω×1nF) ≈ 1.6 MHz，滤除 DAC 输出毛刺。

U1-2 (VOUT B) → 串接 R7 (100 Ω, ±1%) → 接到 U3B-5 (+IN)。U3B-5 到 GND 之间并联 C13 (1 nF/50V NP0)。

U1-10 (VOUT C) → 串接 R8 (100 Ω, ±1%) → 接到 U4A-3 (+IN)。U4A-3 到 GND 之间并联 C16 (1 nF/50V NP0)。

U1-9 (VOUT D) → 串接 R9 (100 Ω, ±1%) → 接到 U4B-5 (+IN)。U4B-5 到 GND 之间并联 C19 (1 nF/50V NP0)。

U1-6 (VOUT E) ~ U1-9 (VOUT H) 未使用 → 悬空不接。软件中向这些通道写入 0x0000。

2.4 DAC8568 去耦

C7 (100 nF/16V X5R)：一端接 U1-16 (VDD)，另一端接 U1-5 (GND)。放置在 U1-16 旁边，距离 < 3 mm。

C8 (10 µF/16V X7R)：一端接 U1-16 (VDD)，另一端接 U1-5 (GND)。稍远放置。

三、OPA2705 #1 (U3) — VOA #1 和 #2 缓冲器

3.1 OPA2705 #1 引脚连接（SOIC-8 封装）

U3-8 (V+) → 接 +5VA。C12 (100 nF/16V X5R) 一端接 U3-8，另一端接 GND，紧靠引脚放置。

U3-4 (V-) → 接 GND（单电源供电）。

3.2 通道 A — VOA #1 缓冲器

U3-3 (+IN, 通道 A 同相输入) → 接 R6 的另一端（来自 DAC8568 VOUT A）。参考 2.3 节。

U3-2 (-IN, 通道 A 反相输入) → 直接连接到 U3-1 (OUT A)。这是单位增益反馈连接（缓冲器）。

U3-1 (OUT A, 通道 A 输出) → 接 R10 (100 Ω, ±1%) 的一端。R10 的另一端分两路：

  • 一路接 C11 (100 nF/50V X7R) 到 GND（形成 16 kHz 一阶低通，后级滤波）
  • 另一路接到输出连接器 J1 (VOA_CTRL1)，控制 VOA #1

3.3 通道 B — VOA #2 缓冲器

U3-5 (+IN, 通道 B 同相输入) → 接 R7 的另一端（来自 DAC8568 VOUT B）。

U3-6 (-IN, 通道 B 反相输入) → 直接连接到 U3-7 (OUT B)。单位增益缓冲器。

U3-7 (OUT B) → 接 R12 (100 Ω, ±1%) 的一端。R12 的另一端：

  • 接 C14 (100 nF/50V X7R) 到 GND
  • 接 J2 (VOA_CTRL2)，控制 VOA #2

C15 (100 nF/16V X5R)：一端接 U3-8(V+)，另一端接 GND（U3 的另一个通道的去耦，与 C12 共用电源引脚）。

四、OPA2705 #2 (U4) — VOA #3 和 #4 缓冲器

4.1 通道 A — VOA #3 缓冲器

U4-8 (V+) → 接 +5VA。C18 (100 nF/16V X5R) 接 U4-8 到 GND。

U4-4 (V-) → GND。

U4-3 (+IN) → 接 R8 的另一端（来自 DAC8568 VOUT C）。

U4-2 (-IN) → 接 U4-1 (OUT A)。单位增益。

U4-1 (OUT A) → 接 R13 (100 Ω, ±1%) → 后分两路：

  • C17 (100 nF/50V X7R) 到 GND
  • J3 (VOA_CTRL3)

C21 (100 nF/16V X5R)：U4-8 去耦，备用。

4.2 通道 B — VOA #4 缓冲器

U4-5 (+IN) → 接 R9 的另一端（来自 DAC8568 VOUT D）。

U4-6 (-IN) → 接 U4-7 (OUT B)。单位增益。

U4-7 (OUT B) → 接 R14 (100 Ω, ±1%) → 后分两路：

  • C20 (100 nF/50V X7R) 到 GND
  • J4 (VOA_CTRL4)

五、OPA381 TIA (U2) — 光功率监测

5.1 OPA381 引脚连接（SOT-23-5 封装）

U2-7 (V+) → 接 +5VA。C23 (100 nF/16V X5R) 一端接 U2-7，另一端接 GND，紧靠引脚。

U2-4 (V-) → 接 GND（单电源供电）。

5.2 光电二极管与 TIA

光电二极管（PD）在光伏模式（零偏）下工作：

PD 阳极 (A) → 接 GND（光伏模式，零偏压）。

PD 阴极 (K) → 接 U2-2 (-IN, 反相输入)。走线尽可能短 (<5 mm)，周围加保护环。

U2-3 (+IN, 同相输入) → 直接接 GND。

R20 (20 kΩ, ±0.1%, 薄膜)：TIA 反馈电阻。一端接 U2-2 (-IN)，另一端接 U2-6 (OUT)。

C22 (10 pF/50V NP0)：TIA 反馈补偿电容。与 R20 并联——一端接 U2-2，另一端接 U2-6。

R20 和 C22 构成 TIA 的反馈网络。时间常数 τ = 20kΩ × 10pF = 200 ns，对应反馈零点 f_z ≈ 796 kHz。详见第 6.1 节的稳定性计算。

5.3 TIA 输出

U2-6 (OUT) → 输出 TIA 电压。VOUT = IPD × 20 kΩ。对于典型 PD（R=0.5 A/W）：

  • 100 µW 光照 → IPD = 50 µA → VOUT = 1.0 V
  • 1 µW 光照 → IPD = 0.5 µA → VOUT = 10 mV

U2-6 输出后经过 C24 (100 nF/16V X5R) AC 耦合电容，进入 OPA2705 #3 的 U5A-3 (+IN)。

六、OPA2705 #3 (U5) — 二级增益 + 抗混叠滤波

6.1 U5A — 二级同相增益（G = 6）

U5-8 (V+) → +5VA。C29 (100 nF/16V X5R) 接 U5-8 到 GND。

U5-4 (V-) → GND。

C24 (100 nF/16V X5R)：一端接 U2-6 (TIA 输出)，另一端接 U5-3 (+IN, U5A 同相输入)。

R21 (100 kΩ, ±1%)：一端接 U5-3 (+IN)，另一端接 GND。C24 + R21 构成高通：f_c = 1/(2π × 100k × 100n) ≈ 16 Hz，滤除 DC 分量。

R23 (10 kΩ, ±0.1%)：一端接 U5-2 (-IN, U5A 反相输入)，另一端接 GND（增益网络的接地端）。

R22 (49.9 kΩ, ±0.1%)：一端接 U5-2 (-IN)，另一端接 U5-1 (OUT A)。

增益 G = 1 + R22/R23 = 1 + 49.9k/10k = 5.99 ≈ 6。TIA 输出 0~1.0V 被放大到 0~6.0V（但被 +5VA 电源轨限制，实际最大输出约 4.98V）。

6.2 U5B — 二阶 Sallen-Key 低通滤波器

U5-1 (OUT A) → 通过 R24 (100 Ω, ±1%) 连接到 U5-5 (+IN, U5B 同相输入) 的路径。

实际上 R24 接的是滤波器网络。路径是：U5-1 → R24 → GND (AC 耦合的参考点？不对，让我重新描述)：

从 U5A 输出到 U5B 的滤波器网络连接：

U5-1 → R24 (100Ω) → R25 (10 kΩ, 0.1%) 的一端。R25 的另一端接 R26 (10 kΩ, 0.1%) 的一端和 C26 (10 nF/50V NP0) 的一端。

R26 的另一端接 U5-5 (+IN, U5B 同相输入) 和 C27 (10 nF/50V NP0) 的一端。

C26 的另一端接 GND。

C27 的另一端接 GND。

U5-6 (-IN, U5B 反相输入) → 直接接 U5-7 (OUT B)（单位增益 Sallen-Key）。

U5-7 (OUT B) → 接 R27 (100 Ω, ±1%) 的一端。R27 的另一端分两路：

  • 接 C28 (100 pF/50V NP0) 到 GND（ADC 采样电容）
  • 接 J5 (ADC_IN) 至 ADC 输入端

滤波器转折频率 f_c = 1/(2π × 10k × 10n) = 1.59 kHz。R25=R26=10kΩ, C26=C27=10nF。Q = 0.5（临界阻尼）。详见第 7 节的计算。

九、完整信号流向描述

9.1 控制路径（MCU → VOA）

  1. MCU 通过 SPI 向 DAC8568 发送 24 位指令。数据从 MCU 的 MOSI 引脚流出，经过 R3 (22Ω) 到达 U1-13 (DIN)。时钟从 MCU 的 SCLK 引脚流出，经过 R2 (22Ω) 到达 U1-12 (SCLK)。CS 由 MCU GPIO 控制，R1 (10kΩ) 上拉到 +3.3VD。
  2. DAC8568 接收到指令后，在指定输出通道（A~D）产生 0~2.5V 模拟电压。该电压从 U1-1/2/10/9 输出，经过 R6~R9 (100Ω) 隔离电阻。
  3. 电压到达 OPA2705 缓冲器（U3A/U3B/U4A/U4B）的同相输入端。运放接成单位增益，输出等于输入电压，但提供了低输出阻抗驱动能力。
  4. 缓冲器输出经过 R10/R12/R13/R14 (100Ω) + C11/C14/C17/C20 (100nF) RC 滤波器（16 kHz 转折频率）。
  5. 滤波后的电压到达 VOA 控制输入连接器 J1~J4，控制 VOA 的衰减量。

9.2 监测路径（PD → MCU）

  1. 光电二极管产生光电流 IPD，流入 OPA381 (U2-2) 的反相输入端。U2-3 (+IN) 接 GND。
  2. TIA 将电流转换为电压：VOUT = IPD × R20 (20kΩ)。反馈电容 C22 (10pF) 保证稳定性。
  3. U2-6 (OUT) 输出经过 C24 (100nF) AC 耦合到 U5A-3 (+IN)。R21 (100kΩ) 将 +IN 偏置到 GND。这构成 16 Hz 高通，滤除 DC 分量。
  4. U5A 放大信号 G = 1 + R22/R23 = 1 + 49.9k/10k = 6 倍。输出电压从 U5-1 输出。
  5. 信号进入 Sallen-Key 低通滤波器：R25 (10k) → R26 (10k) → U5B-5 (+IN)，C26 (10n) 和 C27 (10n) 分别到 GND。
  6. 滤波后的信号从 U5B-7 输出，经过 R27 (100Ω) + C28 (100pF) 到 ADC 输入 J5。ADC 完成数字化后通过 SPI 发送给 MCU。

9.3 MCU 的闭环控制

  1. MCU 从 ADC 读取当前监测光功率（数字量）。
  2. 将测量值与用户设定的目标功率值比较，计算误差。
  3. 运行 PID 算法，计算新的 DAC 输出代码。
  4. 通过 SPI 将新代码写入 DAC8568，调整 VOA 衰减量。
  5. 循环回到第 1 步，更新速率典型值 10~100 Hz。
