# 光电信号链 Glossary

光电检测中涉及的光电转换与模拟信号处理的术语定义。

## 器件与基本概念

**光电二极管 (Photodiode)**
将入射光转换为电流的半导体器件，工作在反偏或光伏模式下。
_Avoid_: 光敏电阻、光电池

**响应度 (Responsivity, R)**
光电二极管将光功率转换为电流的效率，单位 A/W。
_Avoid_: 灵敏度（易混淆，灵敏度通常指最小可检测信号）

**暗电流 (Dark Current, I<sub>D</sub>)**
无光照时流过光电二极管的漏电流。温度每升高 10°C，暗电流大约翻倍。

**PIN 光电二极管**
PIN 结构（P 层-本征层-N 层）的光电二极管，低偏压、低噪声、宽带宽，是光通信测试仪表的默认选择。

**APD (Avalanche Photodiode)**
雪崩光电二极管，利用碰撞电离效应实现内部倍增（M=10~100），等效提高响应度，但引入雪崩噪声 excess noise。

**等效噪声功率 (NEP)**
能产生与 TIA 输出噪声等幅信号所需的最小光功率，单位 W/√Hz。评价接收前端灵敏度的核心指标。

## 电路

**跨阻放大器 (Transimpedance Amplifier, TIA)**
将输入电流转换为输出电压的放大器，核心是运放 + 反馈电阻。光电检测信号链中 I-V 转换的关键电路。
_Avoid_: I-V 转换器（过于泛化）

**反馈电容 (C<sub>f</sub>)**
并联在 TIA 反馈电阻上的电容，用于相位补偿，防止电路自激振荡。
_Avoid_: 补偿电容（需明确是反馈补偿）

**虚短 (Virtual Short)**
理想运放在负反馈下，反相输入端电压近似等于同相输入端电压的特性。TIA 利用此特性将光电流精确引导到反馈支路。

**保护环 (Guard Ring)**
PCB 上围绕高阻抗输入引脚的环形走线，驱动到与输入引脚等电位，用于吸收表面漏电流。GΩ 级 TIA 设计中必不可少。

**自动增益切换 (Auto-Ranging)**
测试仪表根据输入信号强度自动切换 TIA 增益档位以覆盖宽动态范围的技术。通常由 MCU 读取 ADC 结果后控制模拟开关切换反馈电阻。

**可变光衰减器 (Variable Optical Attenuator, VOA)**
在光路中引入可控插入损耗的器件。测试仪表中用于功率校准、接收机灵敏度测试和通道均衡。主流技术为 MEMS VOA。
_Avoid_: 固定衰减器、光开关

**MEMS VOA**
基于微机电系统的 VOA，通过静电或电磁驱动使微镜偏转改变光纤耦合效率实现衰减。测试仪表级默认选择，典型范围 0~60 dB。

**磁光 VOA**
基于法拉第效应的固态 VOA，外加磁场改变磁光晶体偏振旋转角度。响应极快（<0.5 ms），适用于高速功率均衡。

**PDL (Polarization Dependent Loss)**
偏振相关损耗：器件对不同偏振态光的衰减差异。测试级 VOA 要求 PDL <0.05 dB。

**回波损耗 (Return Loss, RL)**
反射光功率与入射光功率之比，单位 dB。测试仪表中高 RL（>50 dB）对防止反射干扰光源稳定至关重要。

**铁氧体磁珠 (Ferrite Bead / Ferrite Chip Bead)**
利用铁氧体材料高频磁损耗将噪声能量转化为热量的被动元件。中高频段表现为电阻性，不同于电感的电抗性反射。精密模拟电路电源去耦的核心器件。
_Avoid_: 等同于电感器

**NiZn 铁氧体**
镍锌铁氧体，初始磁导率 100~1500，适用频率 1 MHz~数百 MHz。贴片磁珠的标准材料。

**MnZn 铁氧体**
锰锌铁氧体，初始磁导率 2000~15000，适用频率 10 kHz~10 MHz。常用于共模扼流圈和低频滤波。

**π 型滤波器**
磁珠串联 + 两侧并联电容到地的滤波器结构。TIA 和 ADC 电源轨的标准去耦方式。

**DCR (DC Resistance)**
磁珠的直流电阻。决定 I²R 自热和电源压降。精密电路中优先选择低 DCR 型号。

**轨到轨 (Rail-to-Rail I/O, RRIO)**
运放的输入和输出都能在接近电源轨的电压范围内工作。OPA2705 的输出可到距电源轨 20 mV 以内。
_Avoid_: 满摆幅（不够精确）

**Sallen-Key 滤波器**
一种基于运放的有源二阶低通滤波器拓扑。优点是元件少、对运放 GBW 要求不高、容易级联。

**抗混叠滤波器 (Anti-Aliasing Filter)**
ADC 前的模拟滤波器，用于抑制超过奈奎斯特频率的信号分量，防止频谱混叠。Sigma-Delta ADC 也需要（防止调制器过载）。

**闭环带宽**
运放在特定增益下的 -3 dB 带宽 = GBW / (1 + R2/R1)。OPA2705 在 10 倍增益下闭环带宽约 100 kHz。

**DAC (数模转换器)**
将数字码转换为模拟电压的器件。在测试仪表中与 ADC 成对出现——ADC 读测量，DAC 写控制。
_Avoid_: 与 ADC 混淆

**R-2R 梯形网络**
DAC 的一种核心架构，使用 R 和 2R 两种阻值组成的电阻网络实现二进制加权分压。优点是单调性固有保证、建立速度快、无纹波。

**双缓冲 (Dual Buffered)**
DAC 内部有两级寄存器（输入寄存器和 DAC 寄存器）。允许先缓存各通道新数据到输入寄存器，然后通过 LDAC 信号同时更新所有输出。

**INL (积分非线性)**
DAC 或 ADC 的实际传输曲线与理想直线之间的最大偏差。DAC8568 的 INL = ±4 LSB 最大。

**DNL (差分非线性)**
DAC 相邻两个数字码之间的实际输出差值与理想 LSB 的偏差。|DNL| < 1 LSB 保证单调性。DAC8568 DNL = ±1 LSB。

**LDAC (Load DAC)**
DAC8568 的控制引脚。拉低时所有 DAC 寄存器同时从输入寄存器加载新数据，实现多通道同步输出更新。

## 通信协议

**SPI (Serial Peripheral Interface)**
同步、全双工、主从式串行总线。4 线制：SCLK、MOSI、MISO、CS。ADC 和 DAC 的默认数字接口。

**CPOL (Clock Polarity)**
SPI 时钟极性。CPOL=0 时 SCLK 空闲为低电平，CPOL=1 时空闲为高电平。

**CPHA (Clock Phase)**
SPI 时钟相位。CPHA=0 时数据在第一个时钟边沿采样，CPHA=1 时在第二个边沿采样。与 CPOL 组合定义 4 个 SPI 模式。

**三态 (High-Z / Tri-state)**
输出引脚的一种状态，呈现高阻抗，相当于引脚与电路断开。多设备共享 MISO 总线时，未选中设备必须进入三态，防止总线冲突。

**片选 (Chip Select, CS)**
SPI 从设备使能信号，低有效。每个从设备独占一根 CS 线。MCU 通过 GPIO 控制 CS 来选择要通信的设备。_Also_: SS (Slave Select), SYNC (TI DAC 系列)

**建立/保持时间 (Setup/Hold Time)**
数字时序参数。建立时间 t_SU = 数据在采样边沿前必须稳定的最短时间。保持时间 t_H = 数据在采样边沿后必须保持稳定的最短时间。违反任一都会导致数据采样错误。
## 数据转换

**量化 (Quantization)**
ADC 将连续的模拟电压映射到离散数字码的过程。量化步长 V<sub>LSB</sub> = V<sub>REF</sub> / 2<sup>N</sup>。

**混叠 (Aliasing)**
信号频率超过奈奎斯特频率（f<sub>s</sub>/2）时，高频成分折叠到低频段产生不可恢复的失真。
_Avoid_: 频谱折叠

**奈奎斯特采样定理**
为了无失真地重建信号，采样频率必须 ≥ 2 × 信号最高频率。

**过采样 (Oversampling)**
以远高于奈奎斯特频率的采样率进行采样，配合数字滤波来提升有效分辨率和 SNR。Σ-Δ ADC 的核心原理。

## 测量与指标

**动态范围**
仪表能准确测量的最大与最小信号功率之比，通常用 dB 表示。光功率计典型值 60~80 dB。

**线性度**
仪表输出（ADC 数字码）与输入光功率之间的线性吻合程度。测试仪表要求远高于通信接收机。

**信噪比 (SNR)**
信号功率与噪声功率之比，用 dB 表示。决定测量精度的下限。
